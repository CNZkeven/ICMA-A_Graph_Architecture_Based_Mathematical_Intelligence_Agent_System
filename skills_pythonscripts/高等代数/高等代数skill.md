## 一、多项式理论

### 核心概念

- **多项式环** $F[x]$：系数取自数域 $F$ 的一元多项式全体
- **整除性**：$f(x) \mid g(x)$ $\iff$ 存在 $h(x)$ 使 $g = f \cdot h$
- **最大公因式**：$\gcd(f,g)$，辗转相除法求得
- **互素**：$\gcd(f,g)=1$，即无非平凡公因式
- **不可约多项式**：次数 $\ge 1$，不能分解为两个次数更低的多项式之积

### 常用公式与定理

| 名称 | 内容 |
|---|---|
| 带余除法 | $f(x) = q(x)g(x) + r(x)$，$\deg r < \deg g$ |
| Bezout定理 | $(f,g)=1 \iff \exists u,v: uf+vg=1$ |
| 因式定理 | $x-a \mid f(x) \iff f(a)=0$ |

### 标准解题步骤

**题型A：多项式除法**
1. 用长除法或综合除法
2. 检查余式次数 < 除式次数
3. 利用余数定理 $r=f(a)$ 快速验证

**题型B：求最大公因式**
1. 执行Euclid辗转相除法
2. 最后一个非零余式归一化即为最大公因式

**题型C：互素证明**
1. 假定存在公因式 $d$（次数 $\ge 1$）
2. 利用整除关系和互素条件推导矛盾

### 常见陷阱

- 忘记检查**余式次数**必须严格小于除式次数
- 最大公因式必须是**首一多项式**
- "不可约"依赖系数域：$x^2+1$ 在 $\mathbb{R}[x]$ 不可约，在 $\mathbb{C}[x]$ 可约

### 做题技巧

- 综合除法每次只操作系数，比长除法快3倍以上
- 求最大公因式时允许对中间余式乘以非零常数简化计算
- 证明互素常用套路：反证法 + 不可约多项式整除性质

## 二、行列式

### 核心概念

- 行列式是 $n\times n$ 矩阵到基域的交错多重线性映射
- Vandermonde行列式：$\prod_{1\le i<j\le n}(x_j-x_i)$
- 特征多项式：$p_A(\lambda)=\det(\lambda I-A)$

### 常用公式

| 名称 | 内容 |
|---|---|
| Vandermonde | $\begin{vmatrix}1&1&1\\x_1&x_2&x_3\\x_1^2&x_2^2&x_3^2\end{vmatrix}=\prod_{i<j}(x_j-x_i)$ |
| 乘法公式 | $\det(AB)=\det(A)\det(B)$ |
| 转置不变 | $\det(A^T)=\det(A)$ |
| 换行变号 | 交换两行，行列式变号 |
| 行倍加不变 | 某行乘以常数加到另一行，行列式不变 |

### 标准解题步骤

**计算行列式：**
1. 观察结构：Vandermonde? 三对角? 分块?
2. 用行变换化简（优先行倍加，不改变行列式值）
3. Laplace展开或利用公式
4. 含参情形用试根法因式分解

### 常见陷阱

- 提取公因子注意：若 $k$ 行每行有公因子 $c$，提取的是 $c^k$
- Laplace展开时符号 $(-1)^{i+j}$ 容易遗漏
- 含参行列式展开后多项式因式分解出错

### 做题技巧

- 优先用行倍加制造零元素
- Vandermonde判别法：相邻两行是否成等比
- 含参行列式用试根法——代入小整数试探零点

## 三、矩阵

### 核心概念

- **逆矩阵**：$A$ 可逆 $\iff \det A\neq0 \iff \operatorname{rank}A=n$
- **秩**：行（列）向量组的极大无关组个数
- **Jordan标准形**：矩阵在相似变换下的最简分块对角形式
- **对角化**：$A$ 可对角化 $\iff$ 每个特征值代数重数=几何重数
- **实对称矩阵**：必可正交对角化

### 常用公式

| 名称 | 内容 |
|---|---|
| 二阶逆矩阵 | $\begin{pmatrix}a&b\\c&d\end{pmatrix}^{-1}=\frac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$ |
| 初等变换求逆 | $[A|I]\xrightarrow{\text{行变换}}[I|A^{-1}]$ |
| Sylvester不等式 | $\operatorname{rank}(A)+\operatorname{rank}(B)-n\le\operatorname{rank}(AB)$ |
| 同时对角化条件 | $AB=BA$ 且均可对角化 $\implies$ 可同时对角化 |

### 标准解题步骤

**求逆矩阵**：构造 $[A|I]$，行变换化 $[I|A^{-1}]$。二阶直接用公式。

**求秩**：行变换化行阶梯形，非零行数即秩。

**对角化判定**：
1. 求特征多项式，找所有特征值
2. 对每个特征值比较代数重数和几何重数
3. $d_i=n-\operatorname{rank}(\lambda_i I-A)$ 为几何重数

### 常见陷阱

- 初等变换求逆只允许**行变换**，不可混用列变换
- 行列式为零不能直接推出秩小于 $n-1$
- Jordan形中Jordan块的排列顺序不唯一，但块结构唯一

## 四、线性方程组

### 核心概念

- 齐次 $Ax=0$：有非零解 $\iff \operatorname{rank}(A)<n$
- 非齐次 $Ax=b$：有解 $\iff \operatorname{rank}(A)=\operatorname{rank}([A|b])$
- 基础解系：解空间的一组基，包含 $n-\operatorname{rank}(A)$ 个线性无关向量
- 通解 = 特解 + 齐次解

### 标准解题步骤

**求基础解系：**
1. 写出系数矩阵，行变换化行最简形（RREF）
2. 确定主元列和自由列
3. 自由变量依次取"1，其余取0"得到基础解系向量

### 做题技巧

- 优先用行变换而非Cramer法则
- 验证：将求得的基础解系代入原方程组检查

## 五、二次型

### 核心概念

- **二次型**：$f(x)=x^TAx$，$A$ 为实对称矩阵
- **配方法**：通过配方化为平方和
- **正定性**：$f(x)>0$（$\forall x\neq0$）$\iff$ 所有顺序主子式 $>0$
- **惯性定理**：正、负惯性指数由二次型唯一确定

### 常用公式

| 判别法 | 内容 |
|---|---|
| Sylvester正定判别 | 所有顺序主子式 $>0$ |
| 惯性定理 | 规范形中正负平方项个数唯一 |
| 配方法 | 集中含某变量的项配方，逐次消去交叉项 |

### 标准解题步骤

**配方法**：
1. 若有平方项，集中配方
2. 若无平方项（所有 $a_{ii}=0$），用 $x_i=y_i+y_j, x_j=y_i-y_j$ 创造平方项
3. 得到 $f=d_1y_1^2+\cdots+d_ry_r^2$，正系数个数为正惯性指数 $p$

**正定性判别**：
1. 计算所有顺序主子式
2. 全正则正定
3. 或计算特征值，全正则正定

### 常见陷阱

- 正定性要求**所有顺序主子式**$>0$，不是所有主子式
- 配方法中线性替换必须**可逆**
- 半正定用顺序主子式 $\ge0$ 只是**必要**非充分条件

## 六、线性空间与线性变换

### 核心概念

- **基与维数**：极大线性无关组，$\dim V$ 唯一确定
- **坐标**：向量在基下的表示系数
- **核(Kernel)**：$\operatorname{Ker}(\sigma)=\{v:\sigma(v)=0\}$
- **像(Image)**：$\operatorname{Im}(\sigma)=\{\sigma(v):v\in V\}$
- **秩-零化度定理**：$\dim\operatorname{Ker}(\sigma)+\dim\operatorname{Im}(\sigma)=\dim V$

### 标准解题步骤

**求坐标**：解 $\beta=x_1\alpha_1+\cdots+x_n\alpha_n$ 得系数向量。

**求核空间**：解 $Ax=0$，解空间的基为核的一组基。

**证明线性无关**：设 $c_1v_1+\cdots+c_kv_k=0$，推导 $c_i=0$。

### 常见陷阱

- 坐标依赖于基的**次序**
- 核的维数 = $n-\operatorname{rank}(A)$，可用秩-零化度定理验证

## 七、欧氏空间

### 核心概念

- **内积**：满足对称性、线性性、正定性的双线性型
- **Schmidt正交化**：逐次减去投影
- **正交变换**：$Q^TQ=I$，保持内积和长度

### 常用公式

| 名称 | 内容 |
|---|---|
| Schmidt正交化 | $\beta_k=\alpha_k-\sum_{i=1}^{k-1}\frac{\langle\alpha_k,\beta_i\rangle}{\langle\beta_i,\beta_i\rangle}\beta_i$ |
| 正交变换性质 | $(Qx)^T(Qy)=x^Ty$，$\det(Q)=\pm1$ |

### 常见陷阱

- 正交化过程中分母 $\langle\beta_i,\beta_i\rangle$ 不可为零
- 正交变换必须是**实**矩阵

## 八、特征值与对角化

### 核心概念

- **特征值/特征向量**：$Av=\lambda v$（$v\neq0$）
- **代数重数**：特征多项式根的重数
- **几何重数**：$\dim\operatorname{Ker}(\lambda I-A)$
- **可对角化** $\iff$ 对所有特征值代数重数=几何重数

### 常用公式

| 名称 | 内容 |
|---|---|
| 迹与行列式 | $\sum\lambda_i=\operatorname{tr}A$，$\prod\lambda_i=\det A$ |
| Cayley-Hamilton | $p_A(A)=0$ |
| 可逆与特征值 | $A$可逆 $\iff 0$不是特征值 |
| 逆阵特征值 | $A^{-1}v=\lambda^{-1}v$ |

### 做题技巧

- 三角矩阵特征值即主对角线元素
- 用迹和行列式验证特征值
- 对角化判定两步走：(1)特征多项式是否完全分解；(2)各特征值代数重数=几何重数?

## 九、通用解题方法论

### 矩阵问题统一框架

高等代数中超过70%的问题可归结为三类：

1. **标准形问题**：等价标准形(秩)、相似标准形(对角化/Jordan)、合同标准形(二次型)
2. **线性方程问题**：增广矩阵→行最简形→自由变量→基础解系
3. **不变性问题**：秩、行列式、迹、特征多项式、最小多项式

**拿到矩阵问题先问三问**：
1. 在什么变换下有意义？（等价/相似/合同？）
2. 在这种变换下什么量不变？
3. 能不能对角化？

### 验证方法速查

| 计算类型 | 验证方法 |
|---|---|
| 逆矩阵 | $A\cdot A^{-1}=I$ |
| 特征值 | $\sum\lambda_i=\operatorname{tr}A$ |
| 特征向量 | 代入 $Av=\lambda v$ |
| 基础解系 | 代入原方程组 |
| Schmidt结果 | 检查两两内积为零 |
| 秩 | 行阶梯形和列阶梯形各算一次 |
