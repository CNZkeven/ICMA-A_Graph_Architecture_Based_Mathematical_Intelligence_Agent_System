## 1. 一阶线性PDE：特征线法

### 1.1 核心概念

一阶线性PDE的一般形式：$a(x,t)u_t + b(x,t)u_x + c(x,t)u = f(x,t)$

特征线法是求解一阶PDE最基本的方法。其思想是：在PDE中寻找**特征曲线**，沿这些曲线PDE退化为ODE。

### 1.2 标准形式与特征方程

对于常系数一阶线性PDE $u_t + c u_x = 0$，特征方程为 $\frac{dx}{dt} = c$，即 $x - ct = \text{const}$。沿特征线，$\frac{d}{dt}u(x(t), t) = u_t + c u_x = 0$，故 $u$ 为常数。解为 $u(x,t) = \phi(x - ct)$，其中 $\phi$ 由初始条件确定。

### 1.3 变系数情况

对于 $u_x + y u_y = 0$，特征方程为 $\frac{dx}{1} = \frac{dy}{y} = \frac{du}{0}$。解得 $y e^{-x} = \text{const}$，$u = \text{const}$（沿特征线）。

### 1.4 标准解题步骤

| 步骤 | 操作 | 示例（$u_t+2u_x=0$） |
|------|------|------|
| 1 | 写出特征方程 $\frac{dx}{dt} = c$ | $\frac{dx}{dt} = 2$ |
| 2 | 解特征线族 | $x - 2t = \xi$ |
| 3 | 沿特征线将PDE转化为ODE | $\frac{du}{dt} = 0$ |
| 4 | 用初值条件确定任意函数 | $u(x,0)=e^{-x^2} \Rightarrow u=e^{-(x-2t)^2}$ |

### 1.5 常见陷阱

- **特征线交叠**：当特征线相交时（如Burgers方程），经典解会破裂，需考虑激波
- **边界条件一致性**：对于有界域，需检查特征线是否从边界进入
- **变系数注意**：特征方程可能依赖于空间变量，需仔细求解常微分方程

### 1.6 做题技巧

- 常系数情形可直接写 $u(x,t) = \phi(x - ct)$，用初值反代
- 对于变系数，将特征方程组写为对称形式 $\frac{dx}{a} = \frac{dt}{b} = \frac{du}{c}$
- 验证方法：回代至原PDE验证解是否成立

## 2. 一阶拟线性PDE：特征线法

### 2.1 核心概念

拟线性PDE的一般形式（无黏Burgers方程）：$u_t + u u_x = 0$。特征线方程为 $\frac{dx}{dt} = u$。

**关键区别**：特征线依赖于解本身，因此特征线不再是直线（除非初值特殊），解沿特征线仍为常数。

### 2.2 解题步骤

1. 写出特征方程组：$\frac{dx}{dt} = u, \frac{du}{dt} = 0$
2. 沿特征线 $u$ 为常数，故 $x = u_0 t + \xi$（特征线为直线）
3. 由初值条件得 $u(x,t) = \phi(x - u t)$
4. 若能从隐式解中显式解出 $u$，则得到显式解

### 2.3 典型题型

| 初值 | 特征线方向 | 解的形式 |
|------|-----------|---------|
| $\phi(x) = \text{const}$ | 平行直线 | $u = \text{const}$ |
| $\phi(x) = x$ | 从原点放射 | $u = x/(1+t)$ |
| $\phi(x) = -x$ | 向原点压缩 | 有限时间破裂 |

### 2.4 常见陷阱

- **破裂时间计算**：当 $\phi'(x) < 0$ 时，特征线会在有限时间内相交，$t_b = -1/\min \phi'(x)$
- **隐式解处理**：拟线性PDE的解往往是隐式的，需判断是否可显式解出

## 3. 二阶PDE分类判别

### 3.1 核心概念

二阶线性PDE一般形式：$A u_{xx} + 2B u_{xy} + C u_{yy} + D u_x + E u_y + F u = G$

**判别式**：$\Delta = B^2 - AC$

| $\Delta$ 符号 | PDE类型 | 典型方程 |
|:---:|:---:|:---:|
| $> 0$ | 双曲型 | 波动方程 $u_{tt} - a^2 u_{xx} = 0$ |
| $= 0$ | 抛物型 | 热传导方程 $u_t - u_{xx} = 0$ |
| $< 0$ | 椭圆型 | 拉普拉斯方程 $u_{xx} + u_{yy} = 0$ |

### 3.2 分类方法

对于方程 $u_{xx} + 4u_{xy} + 4u_{yy} - u_x = 0$：$A = 1$, $B = 2$, $C = 4$，$\Delta = 4 - 4 = 0$，故为**抛物型**。

### 3.3 常见陷阱

- $B$ 对应 $u_{xy}$ 项系数的一半（标准写法中是 $2B u_{xy}$）
- 变系数情形：分类可能依赖于点的位置

---

## 4. 调和函数与拉普拉斯方程

### 4.1 核心概念

**调和函数**：满足拉普拉斯方程 $\Delta u = 0$ 的函数。

基本性质：平均值性质、极值原理（最大值在边界取到）、光滑性（无穷可微）。

### 4.2 验证调和性的方法

直接计算 $u_{xx}$、$u_{yy}$，验证 $u_{xx} + u_{yy} = 0$。

### 4.3 常见陷阱

- 如 $\arctan(y/x)$ 在 $x=0$ 处奇异，需说明定义域
- 极坐标下 $\Delta u = u_{rr} + \frac{1}{r}u_r + \frac{1}{r^2}u_{\theta\theta}$，不要忘记 $1/r$ 项

---

## 5. 波动方程：行波法与分离变量

### 5.1 d'Alembert 公式（全无界问题）

对于柯西问题 $u_{tt} = a^2 u_{xx}$，$u(x,0)=\phi(x)$，$u_t(x,0)=\psi(x)$：

$$u(x,t) = \frac{\phi(x+at) + \phi(x-at)}{2} + \frac{1}{2a}\int_{x-at}^{x+at} \psi(s)\,ds$$

**依赖域**：点 $(x_0, t_0)$ 的解依赖于 $\phi$ 在区间 $[x_0 - at_0, x_0 + at_0]$ 上的值。

### 5.2 分离变量法（有界域）

对于 $[0, L]$ 上齐次 Dirichlet 边界：

1. 设 $u(x,t) = X(x)T(t)$，代入得 $\frac{T''}{a^2 T} = \frac{X''}{X} = -\lambda$
2. 特征值 $\lambda_n = (n\pi/L)^2$，特征函数 $X_n = \sin(n\pi x/L)$
3. 通解：$u = \sum (a_n \cos\frac{n\pi a t}{L} + b_n \sin\frac{n\pi a t}{L})\sin\frac{n\pi x}{L}$
4. 系数由初值 Fourier 展开确定

### 5.3 典型情况速查

| 初值条件 | 解的形式 |
|---------|---------|
| $\phi(x)=\sin(kx)$，$\psi=0$ | $u = \sin(kx)\cos(akt)$ |
| $\phi=0$，$\psi(x)=\sin(kx)$ | $u = \frac{1}{ak}\sin(kx)\sin(akt)$ |

---

## 6. 热传导方程：分离变量与特征展开

### 6.1 分离变量法（有界域）

对于 $u_t = u_{xx}$，齐次 Dirichlet 边界 $u(0,t)=u(L,t)=0$：

1. 设 $u = X(x)T(t)$，得 $T'X = X''T$，$\frac{T'}{T} = \frac{X''}{X} = -\lambda$
2. $T$ 方程：$T' + \lambda T = 0 \Rightarrow T_n(t) = e^{-\lambda_n t}$
3. 特征函数 $X_n = \sin(n\pi x/L)$，$\lambda_n = (n\pi/L)^2$
4. 通解：$u(x,t) = \sum a_n e^{-(n\pi/L)^2 t}\sin(n\pi x/L)$
5. $a_n = \frac{2}{L}\int_0^L \phi(x)\sin(n\pi x/L)\,dx$

### 6.2 关键特征

- **指数衰减**：高阶模态衰减更快
- **光滑性**：$t>0$ 时解无穷可微
- **长期行为**：$t\to\infty$ 时 $u \to 0$（齐次边界下）

---

## 7. 傅里叶变换法

对于全无界热传导问题：

1. 对 $u_t = u_{xx}$ 做傅里叶变换：$\hat{u}_t = -\xi^2 \hat{u}$
2. 解得 $\hat{u}(\xi,t) = \hat{\phi}(\xi) e^{-\xi^2 t}$
3. 逆变换得 $u(x,t) = \phi * G_t$，其中 $G_t(x) = \frac{1}{\sqrt{4\pi t}} e^{-x^2/(4t)}$

**高斯型初值公式**：若 $\phi(x) = e^{-x^2}$，则 $u(x,t) = \frac{1}{\sqrt{1+4t}} e^{-\frac{x^2}{1+4t}}$。

---

## 8. 拉普拉斯方程：圆域与极坐标

极坐标下 $\Delta u = u_{rr} + \frac{1}{r}u_r + \frac{1}{r^2}u_{\theta\theta} = 0$。

圆域 Dirichlet 问题 $u(1,\theta) = f(\theta)$：

1. 设 $u = R(r)\Theta(\theta)$，得 $\Theta_n(\theta) = \cos(n\theta), \sin(n\theta)$
2. $R_n(r) = r^n$（有界性条件去掉 $r^{-n}$）
3. 通解：$u = \frac{a_0}{2} + \sum r^n(a_n\cos n\theta + b_n\sin n\theta)$
4. 系数由边界 Fourier 展开确定

**特殊情形**：$u(1,\theta)=\cos(2\theta) \Rightarrow u(r,\theta)=r^2\cos(2\theta)$（仅 $n=2$ 模态）

---

## 9. 非齐次边界与稳态分析

设 $u(x,t) = v(x) + w(x,t)$，其中 $v$ 为稳态解（$v_{xx}=0$），$w$ 为暂态解。

以 $u_t = u_{xx}$，$u(0,t)=0,u(1,t)=1,u(x,0)=0$ 为例：
- 稳态 $v(x) = x$
- 暂态 $w$ 满足齐次方程+齐次边界，$w(x,0) = -x$
- $w(x,t) = \sum \frac{2(-1)^n}{n\pi} e^{-n^2\pi^2 t}\sin(n\pi x)$

---

## 10. 半无界问题：反射法

半无界 $x > 0$ 通过延拓转化为全无界：
- **Dirichlet 边界 $u(0,t)=0$** → 奇延拓
- **Neumann 边界 $u_x(0,t)=0$** → 偶延拓

奇延拓后在全无界上应用 d'Alembert 公式，再限制回 $x\ge0$。

---

## 11. 非齐次方程：特征展开法

对于 $u_t = u_{xx} + f(x,t)$，将解和源项都用特征函数展开：

$$u = \sum T_n(t)\sin\frac{n\pi x}{L}, \quad f = \sum f_n(t)\sin\frac{n\pi x}{L}$$

$T_n$ 满足 $T_n' + (n\pi/L)^2 T_n = f_n(t)$，用积分因子法求解。

**源项恰为特征函数时**：$f(x,t)=\sin(2x)$ 对应 $n=2$，$T_2' + 4T_2 = 1$，$T_2 = \frac{1}{4}(1-e^{-4t})$。

---

## 12. 能量方法与唯一性

### 12.1 波动方程能量积分

$E(t) = \frac{1}{2}\int_0^L (u_t^2 + a^2 u_x^2)\,dx$

对两解之差 $w$，$E'(t) = a^2[w_x w_t]_0^L = 0$（齐次边界），故 $E$ 为常数。由 $E(0)=0$ 得 $w\equiv0$，唯一性得证。

### 12.2 拉普拉斯方程能量方法

Green 第一恒等式：$\int_\Omega u\Delta v\,dx = \int_{\partial\Omega} u\frac{\partial v}{\partial n}\,dS - \int_\Omega \nabla u\cdot\nabla v\,dx$

代入 $\Delta u=0$ 和 $u|_{\partial\Omega}=0$，得 $\int_\Omega |\nabla u|^2 dx = 0 \Rightarrow \nabla u \equiv 0 \Rightarrow u\equiv0$。

---

## 13. 极值原理

热传导方程的极值原理：解的最大（小）值在抛物边界取到。

证明方法：构造 $v = u - \epsilon t$，则 $v_t - v_{xx} = -\epsilon < 0$。若 $v$ 在内部取最大值，则 $v_t \ge 0$, $v_{xx} \le 0$，矛盾。故最大值在边界取到，令 $\epsilon\to0$ 即得。

---

## 14. 格林函数与对称性

格林函数 $G(x,\xi)$ 满足 $\Delta_x G(x,\xi) = \delta(x-\xi)$（在 $\Omega$ 内），$G(x,\xi) = 0$（$x\in\partial\Omega$）。

对称性 $G(x,\xi) = G(\xi,x)$ 通过 Green 第二恒等式证明：挖去两奇点小球，应用恒等式，令小球半径趋于零。

---

## 15. 依赖域与传播特征

- 一阶方程 $u_t + c u_x = 0$：依赖域退化为单点 $\{x_0 - ct_0\}$
- 二阶波动方程 $u_{tt} = a^2 u_{xx}$：依赖域为区间 $[x_0-at_0, x_0+at_0]$
- 热传导方程：传播速度无限（与波动方程形成对比）

---

## 16. 稳定性估计

热传导方程 $L^2$ 稳定性：$E(t) = \frac{1}{2}\int_0^L u^2 dx$，

$E'(t) = \int u u_{xx} dx = -\int u_x^2 dx \le 0$，故 $E(t) \le E(0)$。

由此得 $\int_0^L u^2(x,t)\,dx \le \int_0^L \phi^2(x)\,dx$，即解在 $L^2$ 意义下连续依赖于初值。

## 17. 通用解题方法论

### 17.1 PDE类型识别决策树

```
PDE
├─ 一阶
│  ├─ 线性: u_t + c u_x = 0 → 特征线法
│  └─ 拟线性: u_t + u u_x = 0 → 特征线法（隐式解）
└─ 二阶
   ├─ Δ > 0 (双曲) → 波动方程
   │  ├─ 全无界 → d'Alembert公式
   │  ├─ 有界域 → 分离变量法
   │  └─ 半无界 → 反射法(延拓)
   ├─ Δ = 0 (抛物) → 热传导方程
   │  ├─ 全无界 → Fourier变换/热核卷积
   │  ├─ 有界齐次边界 → 分离变量法
   │  ├─ 有界非齐次边界 → 稳态+暂态分解
   │  └─ 有源项 → 特征展开法
   └─ Δ < 0 (椭圆) → 拉普拉斯方程
      ├─ 矩形域 → 分离变量法
      ├─ 圆域 → 极坐标分离变量
      └─ 一般域 → 格林函数/变分法
```

### 17.2 验证结果的通用方法

1. **代入检验**：将所得解代入原PDE验证
2. **初边界条件检验**：验证全部初始和边界条件
3. **特殊极限检验**：$t\to0$ 趋于初值，$t\to\infty$ 趋于稳态
4. **对称性检验**：若问题具有对称性，解也应具有

### 17.3 常见错误汇总

| 错误类型 | 典型表现 | 纠正方法 |
|---------|---------|---------|
| 特征线方向错误 | 一阶方程方向反了 | 检查 $dx/dt = c$ 而非 $-c$ |
| 特征值符号错误 | 分离变量时 $\lambda$ 符号反了 | 检查 $X''/X = -\lambda$ |
| 归一化因子遗漏 | Fourier系数缺 $2/L$ | 重新推导 |
| 边界条件未齐次化 | 直接对非齐次边界用分离变量 | 先稳态分解 |
| 奇延拓符号错误 | 反射波相位错误 | Dirichlet→奇延拓，Neumann→偶延拓 |
