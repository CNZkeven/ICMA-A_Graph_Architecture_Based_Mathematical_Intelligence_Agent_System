## 1. Lebesgue 测度基础

### 核心定义与定理

| 概念 | 定义 |
|------|------|
| **Lebesgue 外测度** | \( m^*(E) = \inf\{ \sum |I_k| : E \subset \bigcup I_k \} \)，其中 \(I_k\) 为开区间 |
| **Lebesgue 可测集** | \(E\) 满足 Carathéodory 条件：\(\forall A\subset \mathbb{R}^n,\; m^*(A) = m^*(A\cap E) + m^*(A\setminus E)\) |
| **零测集** | 外测度为 0 的集合，如可数集、Cantor 集 |
| **可测集性质** | \(\sigma\)-代数性质：可数并、可数交、补集封闭 |
| **正则性** | \(\forall E\) 可测，\(m(E) = \inf\{ m(U): E\subset U,\; U\text{开} \} = \sup\{ m(K): K\subset E,\; K\text{紧} \}\) |

### 核心公式

- 区间测度：\(m([a,b]) = m((a,b)) = b-a\)
- 可数可加性：若 \(\{E_k\}\) 两两不交可测，则 \(m(\bigcup E_k) = \sum m(E_k)\)
- 可数集的测度为零：\(m(\mathbb{Q}) = 0\)

### 标准化解题流程

1. 判断集合是否可数 → 零测
2. 利用区间可加性分割集合
3. 利用 \(\sigma\)-代数性质将复杂集合拆解为简单集

### 考试高频陷阱

- **不可测集存在**：依赖于选择公理，考试中遇到的集合通常是可测的
- **零测集不一定是可数集**：Cantor 集是不可数的零测集
- **测度非负但可无限**：开集测度可能为无穷

## 2. 可测函数

### 核心定义与定理

| 概念 | 定义 |
|------|------|
| **可测函数** | \(f: X \to \mathbb{R}\) 满足 \(\forall a\in\mathbb{R},\; \{x: f(x) > a\}\) 可测 |
| **简单函数** | 形如 \(f = \sum_{i=1}^n a_i \chi_{E_i}\)，其中 \(E_i\) 可测、两两不交 |
| **几乎处处** | 除零测集外处处成立 |
| **Dirichlet 函数** | \(\chi_{\mathbb{Q}}(x)\)，有理点为 1，无理点为 0 |
| **Egorov 定理** | a.e. 收敛 \(\Rightarrow\) 在测度任意大的紧集上一致收敛 |

### 核心性质

- 可测函数的极限（逐点、a.e.）仍可测
- 连续函数 \(\Rightarrow\) 可测函数
- 可测函数可用简单函数序列逼近

### 考试高频陷阱

- **Riemann 不可积但 Lebesgue 可积**：Dirichlet 函数是经典例子
- **几乎处处相等的函数视为同一**：在 Lp 空间中，a.e. 相等的函数等价

## 3. Lebesgue 积分

### 核心定义

\[
\int f\,dm = \sup\{ \int g\,dm : 0\le g\le f,\; g\text{ 为简单函数} \}
\]

对一般可测函数：\(f = f^+ - f^-\)，其中 \(f^+ = \max(f,0)\)，\(f^- = \max(-f,0)\)。

### 核心公式

- 对非负可测函数：\(\int f\,dm = \lim_{n\to\infty} \int \phi_n\,dm\)（\(\phi_n\) 单调逼近 \(f\)）
- Riemann 可积函数也是 Lebesgue 可积，积分值相等
- **Newton-Leibniz**：若 \(f\) 绝对连续，则 \(f(b)-f(a) = \int_a^b f'\,dm\)

### 标准化解题流程

1. 检查函数是否可测
2. 判断非负性 → 用单调收敛或简单函数逼近
3. 对有正有负函数：拆分正部负部分别积分
4. 利用 Riemann 积分计算（对 Riemann 可积函数）

### 考试高频陷阱

- **Lebesgue 积分不处理广义积分**：\(\int_0^\infty \frac{\sin x}{x}\,dx\) 在 Lebesgue 意义下不可积
- **Lebesgue 积分是绝对收敛积分**：\(\int f\) 存在当且仅当 \(\int |f|\) 存在

## 4. 简单函数积分

### 核心定义

\[
f(x) = \sum_{i=1}^n a_i \chi_{E_i}(x),\quad \int f\,dm = \sum_{i=1}^n a_i m(E_i)
\]

### 标准化解题流程

1. 识别简单函数的取值区间划分
2. 计算每个区间的测度
3. 求和：\(\sum\) (取值 \(\times\) 测度)

### 典型例题（idx=2）

- \(f(x) = 2\) 在 \([0,1)\)，\(f(x)=3\) 在 \([1,2]\)
- \(\int f\,dm = 2\times m([0,1)) + 3\times m([1,2]) = 2\times 1 + 3\times 1 = 5\)

## 5. Lp 空间与范数

### 核心定义

\[
\|f\|_p = \left( \int |f|^p\,dm \right)^{1/p},\quad 1\le p < \infty
\]
\[
\|f\|_\infty = \operatorname{ess\,sup} |f| = \inf\{ M > 0 : |f|\le M \text{ a.e.} \}
\]

### 核心公式

- \(\|f\|_p^p = \int |f|^p\,dm\)
- \(\lim_{p\to\infty} \|f\|_p = \|f\|_\infty\)（当 \(f\in L^\infty \cap L^p\) 对某个 \(p\)）
- \(L^p \subset L^q\) 对有限测度空间且 \(p\ge q\)

### 标准化解题流程

1. 确定测度空间类型（有限 / \(\sigma\)-有限）
2. 计算积分：\(\int |f|^p\,dm\)
3. 开 \(p\) 次方得 \(\|f\|_p\)
4. 取本质确界得 \(\|f\|_\infty\)

### 考试高频陷阱

- **Lp 范数依赖于测度空间**：不同测度下同一函数范数不同
- **p 越小对函数峰值越敏感**：\(p\to\infty\) 时只关心最大值

## 6. 单调收敛定理（Levi 引理）

### 核心定理

设 \(\{f_n\}\) 是非负可测函数单调递增列（\(0\le f_1\le f_2\le \cdots\)），则
\[
\lim_{n\to\infty} \int f_n\,dm = \int \lim_{n\to\infty} f_n\,dm
\]

### 标准化解题流程

1. 验证 \(f_n\) 非负可测
2. 验证 \(f_n\) 单调递增
3. 求逐点极限 \(f = \lim f_n\)
4. 交换极限与积分

### 典型例题（idx=4, idx=12）

- **idx=4**：\(f_n(x) = \frac{nx}{1+n^2x^2}\)，逐点收敛于 0，\(\lim \int f_n = 0\)
- **idx=12**：\(-\ln x = \lim_{n\to\infty} -\ln(\max(x,1/n))\)，\(\int_0^1 -\ln x\,dx = 1\)

---

## 7. 控制收敛定理

### 核心定理

设 \(\{f_n\}\) 是可测函数，逐点（a.e.）收敛于 \(f\)，且存在非负可积函数 \(g\) 使得 \(|f_n|\le g\) a.e.，则
\[
\lim_{n\to\infty} \int f_n\,dm = \int f\,dm
\]

### 标准化解题流程

1. 找控制函数 \(g\)：满足 \(|f_n|\le g\)，且 \(\int g < \infty\)
2. 验证逐点极限 \(f\)
3. 应用 DCT：\(\lim\int f_n = \int f\)

### 典型例题（idx=5）

- \(f_n(x) = \frac{\sin(nx)}{n(1+x^2)}\)
- 控制函数：\(|f_n| \le \frac{1}{1+x^2} \in L^1([0,\infty))\)
- 逐点极限为 0，故 \(\lim \int f_n = 0\)

---

## 8. Fatou 引理

### 核心定理

设 \(\{f_n\}\) 是非负可测函数列，则
\[
\int \liminf_{n\to\infty} f_n\,dm \le \liminf_{n\to\infty} \int f_n\,dm
\]

### 标准化解题流程

1. 验证 \(f_n\) 非负可测
2. 计算 \(\liminf f_n\) 的积分
3. 计算 \(\liminf \int f_n\)
4. 比较两者，验证不等式

### 典型例题（idx=6）

- \(f_n(x) = n\chi_{(0,1/n]}(x)\)
- \(\liminf f_n = 0\) a.e.，\(\int \liminf f_n = 0\)
- \(\int f_n = n\cdot(1/n) = 1\)，\(\liminf \int f_n = 1\)
- 严格不等式：\(0 < 1\)

### 考试高频陷阱

- **反向不等式不成立**：Fatou 引理只有 \(\le\) 方向
- **等号不成立条件**：被积函数需要一致可积才能交换极限与积分

## 9. Lp 范数不等式

### 核心公式

**单调性**：在有限测度空间上，若 \(1\le p < q \le \infty\)，则
\[
\|f\|_p \le \|f\|_q \cdot m(X)^{1/p - 1/q}
\]
特别地，若 \(m(X)=1\)（概率测度），则 \(\|f\|_p \le \|f\|_q\)。

### 典型例题（idx=7）

- \(f(x) = x^{1/3}\) 在 \([0,1]\) 上
- \(\|f\|_1 = \int_0^1 x^{1/3}dx = 3/4 = 0.75\)
- \(\|f\|_3 = (\int_0^1 x\,dx)^{1/3} = (1/2)^{1/3} \approx 0.794\)
- 验证：\(0.75 \le 0.794\)

---

## 10. Fubini 定理

### 核心定理

设 \(f\) 在乘积空间 \(X\times Y\)（\(\sigma\)-有限测度）上非负可测或可积，则
\[
\int_{X\times Y} f\,d(\mu\times\nu) = \int_X \left( \int_Y f(x,y)\,d\nu(y) \right) d\mu(x) = \int_Y \left( \int_X f(x,y)\,d\mu(x) \right) d\nu(y)
\]

### 标准化解题流程

1. 验证被积函数非负或绝对可积
2. 按先 x 后 y 顺序计算累次积分
3. 按先 y 后 x 顺序验证结果相同

### 典型例题（idx=10）

- \(\iint_{[0,1]^2} xy\,dm_2 = \int_0^1 \int_0^1 xy\,dx\,dy = 1/4\)

### 考试高频陷阱

- **Fubini 不成立时**：\(\iint \frac{x^2 - y^2}{(x^2+y^2)^2}\) 在 \([0,1]^2\) 上两次积分顺序交换结果不同
- **必须验证可积条件**：非负可测或 \(\iint |f| < \infty\)

## 11. 绝对连续函数与 Newton-Leibniz 公式

### 核心定义

**绝对连续函数**：\(\forall \varepsilon>0\)，\(\exists \delta>0\) 使得对任意有限个不交区间 \(\{(a_k,b_k)\}\)，若 \(\sum(b_k-a_k)<\delta\)，则 \(\sum |f(b_k)-f(a_k)|<\varepsilon\)。

### 核心定理

\(f\) 绝对连续 \(\iff\) \(f\) a.e. 可导，\(f'\in L^1\)，且
\[
f(x) - f(a) = \int_a^x f'(t)\,dt
\]

### 标准化解题流程

1. 检查 \(f\) 是否 Lipschitz / 绝对连续
2. 计算导数 \(f'\)
3. 验证 \(\int_a^b f' = f(b)-f(a)\)

### 典型例题（idx=9）

- \(f(x)=x^2\) 在 \([0,1]\) 上绝对连续
- \(f'(x)=2x\)，\(\int_0^1 2x\,dx = 1 = f(1)-f(0)\)

## 12. Radon-Nikodym 定理

### 核心定理

若 \(\nu \ll \mu\)（\(\nu\) 关于 \(\mu\) 绝对连续），则存在可测函数 \(g\) 使得
\[
\nu(E) = \int_E g\,d\mu,\quad \forall E\text{ 可测}
\]
记 \(g = \frac{d\nu}{d\mu}\)。

### 标准化解题流程

1. 验证 \(\nu \ll \mu\)（\(\mu(E)=0 \Rightarrow \nu(E)=0\)）
2. 从 \(\nu\) 的定义直接读取 Radon-Nikodym 导数
3. 对具体集合计算测度值

### 典型例题（idx=13）

- \(\nu(E) = \int_E 2x\,d\mu\)，直接得 \(\frac{d\nu}{d\mu} = 2x\)
- \(\nu([0,1]) = \int_0^1 2x\,dx = 1\)

## 13. Chebyshev 不等式

### 核心公式

设 \(f\ge 0\) 可测，\(\lambda > 0\)，则
\[
m(\{x: f(x) \ge \lambda\}) \le \frac{1}{\lambda} \int f\,dm
\]

### 标准化解题流程

1. 确定 \(f\) 和 \(\lambda\)
2. 计算左端：找出 \(\{f\ge \lambda\}\) 并求其测度
3. 计算右端：\(\frac{1}{\lambda}\int f\)
4. 比较验证

### 典型例题（idx=14）

- \(f(x)=x^2\)，\(\lambda = 1/4\)
- 左端：\(m([1/2,1]) = 1/2\)
- 右端：\(4 \times \int_0^1 x^2\,dx = 4 \times 1/3 = 4/3\)
- 验证：\(1/2 \le 4/3\)

---

## 14. Holder 不等式

### 核心公式

设 \(p,q > 1\)，\(\frac{1}{p} + \frac{1}{q} = 1\)，则
\[
\int |fg|\,d\mu \le \|f\|_p \|g\|_q
\]

当 \(p=q=2\) 时即为 Cauchy-Schwarz 不等式。

### 标准化解题流程

1. 确定共轭指数对 \((p,q)\)
2. 计算 \(\|f\|_p = (\int |f|^p)^{1/p}\)
3. 计算 \(\|g\|_q = (\int |g|^q)^{1/q}\)
4. 计算 \(\int |fg|\)，验证不等式

### 典型例题（idx=15）

- \(f(x)=x\)，\(g(x)=1-x\)，\(p=q=2\)
- \(\int fg = \int_0^1 x(1-x)dx = 1/6\)
- \(\|f\|_2 = \sqrt{1/3}\)，\(\|g\|_2 = \sqrt{1/3}\)，乘积 \(= 1/3\)
- 验证：\(1/6 \le 1/3\)

---

## 15. Minkowski 不等式

### 核心公式

设 \(1\le p < \infty\)，则
\[
\|f+g\|_p \le \|f\|_p + \|g\|_p
\]

### 标准化解题流程

1. 确定 \(p\)
2. 计算 \(\|f\|_p\) 和 \(\|g\|_p\)
3. 计算 \(\|f+g\|_p\)
4. 验证三角不等式

### 典型例题（idx=16）

- \(f(x)=x\)，\(g(x)=x^2\)，\(p=2\)
- \(\|f+g\|_2 = (\int_0^1 (x+x^2)^2 dx)^{1/2} = \sqrt{31/30} \approx 1.0165\)
- \(\|f\|_2 = \sqrt{1/3} \approx 0.5774\)，\(\|g\|_2 = \sqrt{1/5} \approx 0.4472\)
- 验证：\(1.0165 \le 1.0246\)

## 16. 收敛模式：a.e. 收敛与 Lp 收敛

### 核心定义

| 收敛类型 | 定义 |
|---------|------|
| **几乎处处收敛** | \(f_n(x) \to f(x)\) 对几乎所有 \(x\) |
| **Lp 收敛** | \(\|f_n - f\|_p \to 0\) |
| **依测度收敛** | \(\forall\varepsilon>0,\; m(\{|f_n-f|\ge\varepsilon\}) \to 0\) |

### 关系图

\[
\text{Lp 收敛} \xRightarrow{m(X)<\infty} \text{依测度收敛} \xRightarrow{\text{子列}} \text{a.e. 收敛}
\]

### 标准化解题流程

1. 判断 a.e. 收敛：对每个 \(x\) 取极限（注意零测集）
2. 计算 \(\|f_n\|_p\) 判断 Lp 收敛
3. 若 a.e. 收敛 + 控制条件 \(\Rightarrow\) Lp 收敛

### 典型例题（idx=17）

- \(f_n(x) = x^n\) 在 \([0,1]\) 上
- a.e. 收敛于 0（在 \(x=1\) 处不收敛，但单点零测）
- \(\|f_n\|_1 = \int_0^1 x^n dx = 1/(n+1) \to 0\)，L¹ 收敛

## 17. 卷积

### 核心定义

\[
(f * g)(t) = \int_{\mathbb{R}} f(s) g(t-s)\,ds
\]

### 核心性质

- 卷积交换：\(f*g = g*f\)
- 特征函数卷积：\(\chi_{[0,1]} * \chi_{[0,1]}(t)\) 为三角形函数

### 标准化解题流程

1. 确定卷积定义式
2. 根据支撑集确定被积函数非零范围
3. 分段计算积分
4. 得到最终表达式

### 典型例题（idx=18）

- \(h(t) = \chi_{[0,1]} * \chi_{[0,1]}(t)\)
- \(h(t) = 
  \begin{cases}
  0, & t \le 0 \\
  t, & 0 \le t \le 1 \\
  2-t, & 1 \le t \le 2 \\
  0, & t \ge 2
  \end{cases}\)
- \(h(1) = 1\)

## 18. 测度构造与绝对连续

### 核心定义

**通过密度函数构造测度**：
\[
\mu(E) = \int_E f\,dm,\quad f\ge 0\text{ 可测}
\]
此时 \(\mu \ll m\)，\(\frac{d\mu}{dm} = f\)。

### 标准化解题流程

1. 由 \(f\) 定义测度 \(\mu\)
2. 验证绝对连续性：若 \(m(E)=0\) 则 \(\mu(E)=0\)
3. 计算具体集合的测度值

### 典型例题（idx=19）

- \(\mu(E) = \int_E x\,dm\)
- \(\frac{d\mu}{dm} = x\)
- \(\mu([0,1]) = \int_0^1 x\,dx = 1/2\)
- 对 Cantor 集 \(C\)，\(m(C)=0\)，由绝对连续性得 \(\mu(C)=0\)

## 19. 通用解题方法论

### 题型识别方法

| 题目特征 | 对应方法 |
|---------|---------|
| 求积分值，函数连续/Riemann 可积 | 直接用 Riemann 积分计算 |
| 被积函数有参数 \(n\)，求极限 | MCT / DCT / Fatou |
| 求范数 | Lp 积分定义 |
| 证明不等式 | Chebyshev / Holder / Minkowski |
| 二重积分交换次序 | Fubini 定理 |
| 判断可积性 | 可测性 + 绝对可积性检查 |
| 求 Radon-Nikodym 导数 | 从测度定义中提取密度函数 |
| 卷积计算 | 支撑集分段讨论 |

### 解题工具选择

1. **先看函数是否非负**：非负可测 → MCT 或简单函数逼近
2. **有界且测度有限**：有界收敛定理（BCT）
3. **需要控制函数**：DCT（寻找 \(L^1\) 控制函数）
4. **只需不等式**：Fatou 引理
5. **具体函数计算**：Riemann 积分 + 保持性质

### 结果核验思路

- **量纲/范围检查**：积分值应在合理范围内
- **对称性检查**：若函数有对称性，积分值也应体现
- **不等式自洽**：若 A ≤ B 且 B ≤ A，则等号成立
- **极限交换验证**：同时计算逐点极限和积分极限比对
