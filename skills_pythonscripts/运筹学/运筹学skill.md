# 运筹学解题技能手册

> 基于20道典型习题（14计算+6证明），系统梳理运筹学核心解题技能。

---

## 模块一：线性规划

### 核心概念

决策变量、目标函数、约束条件、可行域、基可行解、对偶问题、影子价格。

### 图解法（两个变量）

1. 画出约束直线，标不等式方向，确定可行域
2. 画目标函数等值线，沿改进方向平移
3. 交点为最优解

### 单纯形法

1. 引入松弛变量化为标准形
2. 确定初始基可行解
3. 计算检验数$\sigma_j=c_j-\mathbf{c}_B B^{-1}\mathbf{a}_j$
4. 最优性：所有$\sigma_j\le0$（max）；否则最大正$\sigma_j$进基
5. 最小比值规则$\theta=\min\{b_i/a_{ik}\mid a_{ik}>0\}$定出基
6. 主元消去，重复

### 对偶问题构建

原max约束$\le$↔对偶变量$\ge0$；原变量$\ge0$↔对偶约束$\ge$。

### 弱对偶定理

$c^T x\le b^T y$对任意原/对偶可行解成立（max原问题）。

## 模块二：运输问题

### 最小元素法

1. 找最小$c_{ij}$，分配$\min(a_i,b_j)$
2. 更新供需，划去已满行/列
3. 重复至所有供需满足
4. 基变量应为$m+n-1$个

### 位势法

- 基变量满足$u_i+v_j=c_{ij}$，令$u_1=0$解出所有位势
- 非基变量检验数$\sigma_{ij}=c_{ij}-u_i-v_j$
- 全$\ge0$则最优（min问题）

## 模块三：整数规划

**LP松弛下界**：$z_{LP}^*\le z_{IP}^*$（min问题），因$X_{IP}\subset X_{LP}$。

**分支定界法**：求解LP松弛→选非整数变量分支→对子问题求解→剪枝。

## 模块四：非线性规划

### KKT条件

对$\min f(\mathbf{x})$ s.t. $g_i(\mathbf{x})\le0, h_j(\mathbf{x})=0$：
$$\nabla f+\sum\mu_i\nabla g_i+\sum\lambda_j\nabla h_j=0,\quad \mu_i g_i=0,\quad \mu_i\ge0$$

### KKT充分性（凸规划）

若$f$凸、$g_i$凸、$h_j$仿射，则KKT点就是全局最优解。

### 解题步骤

1. 写出Lagrange函数
2. 列平稳性、原始可行、对偶可行、互补松弛条件
3. 对不等式约束分类讨论（$\mu_i=0$或$g_i=0$）
4. 求解并验证

## 模块五：动态规划

### 逆序递推基本方程

$$f_k(s_k)=\min/\max_{u_k}\{v_k(s_k,u_k)+f_{k+1}(s_{k+1})\},\quad f_{n+1}=0$$

### 解题步骤

1. 划分阶段，定义状态和决策
2. 写出状态转移方程和阶段指标
3. 建立递推方程
4. 从最后阶段向前递推
5. 顺推得最优策略

## 模块六：图论与网络优化

### Dijkstra最短路

1. 初始化$d(s)=0$，其余$\infty$
2. 选未标号最小$d(u)$，永久标号
3. 更新邻接点：$d(v)=\min\{d(v),d(u)+w(u,v)\}$
4. 重复至目标点标号

### Ford-Fulkerson最大流

1. 零流初始
2. 标号法找增广路径（前向弧未饱和/后向弧有流）
3. 汇点标号→沿路径增流
4. 无增广路为止→当前流最大

### 最大流最小割定理

任意流值$\le$任意割容量。算法终止时残余网络中$s$可达点集$S$构成最小割，流值=割容量。

## 模块七：排队论

### M/M/1（$\rho=\lambda/\mu<1$）

| 指标 | 公式 |
|------|------|
| $P_0$ | $1-\rho$ |
| $L$ | $\rho/(1-\rho)$ |
| $L_q$ | $\rho^2/(1-\rho)$ |
| $W$ | $L/\lambda=1/(\mu-\lambda)$ |
| $W_q$ | $L_q/\lambda$ |

### M/M/c（$\rho=\lambda/(c\mu)<1$）

$$P_0=\left[\sum_{n=0}^{c-1}\frac{(c\rho)^n}{n!}+\frac{(c\rho)^c}{c!(1-\rho)}\right]^{-1}$$
$$L_q=\frac{(c\rho)^c\rho}{c!(1-\rho)^2}P_0,\quad L=L_q+c\rho$$

## 模块八：存储论

### EOQ模型

$$TC(Q)=\frac{KD}{Q}+\frac{hQ}{2},\quad Q^*=\sqrt{\frac{2KD}{h}},\quad TC^*=\sqrt{2KDh}$$

**推导**：$TC'(Q)=0\Rightarrow Q^*$，$TC''(Q)=2KD/Q^3>0$（严格凸）。

---

## 模块九：博弈论

### 2×2混合策略

设行玩家策略$(p,1-p)$，列玩家$(q,1-q)$。无差异原理：令两条期望线相等解$p,q$。

### 2×n图解法

1. 行玩家各纯策略对应期望$v_i(p)$直线
2. 取下包络线（列玩家最小化）
3. 包络线最高点→最优$p$和博弈值$v$

### 鞍点条件

$(i^*,j^*)$为鞍点$\iff a_{ij^*}\le a_{i^*j^*}\le a_{i^*j}$对所有$i,j$成立。

## 模块十：决策论

### EMV准则

$$EMV_i=\sum_j p_j r_{ij}$$

选择$EMV$最大的方案。

### 决策树

矩形节点=决策节点（选分支），圆形节点=概率节点（取期望），逆推从叶到根。

| 方法 | 适用 | 操作 |
|------|------|------|
