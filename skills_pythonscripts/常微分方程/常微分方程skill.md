## 第一部分：知识模块

### 模块1：一阶可分离变量方程

**核心概念：** 形如 $\frac{dy}{dx}=f(x)g(y)$ 的方程，可以将含 $x$ 和含 $y$ 的项分别移到等号两边。

**标准形式：** $\frac{dy}{dx}=f(x)g(y)$

**标准解题步骤：**
1. **分离变量：** 将方程写为 $\frac{dy}{g(y)}=f(x)dx$
2. **积分：** 两边同时积分 $\int\frac{dy}{g(y)}=\int f(x)dx + C$
3. **解出y：** 如果可能，显式解出 $y=y(x,C)$
4. **代入初值：** 若有初值条件 $y(x_0)=y_0$，代入确定常数 $C$

**常见陷阱：**
- 分离变量时若 $g(y)=0$ 有根，这些根可能是奇解，需要单独讨论
- 积分后不要忘记加常数 $C$
- 绝对值符号处理：$\int\frac{dy}{y}=\ln|y|$，但若已知 $y>0$ 可去掉绝对值

**做题技巧：**
- 先判断是否可分离：右边能否写成 $f(x)\cdot g(y)$ 的形式
- 积分后优先用初值条件确定常数，避免带着 $C$ 做过多的代数运算

**对应习题：** idx 0

### 模块2：一阶线性微分方程

**核心概念：** 形如 $\frac{dy}{dx}+P(x)y=Q(x)$ 的方程。

**核心公式：** 积分因子 $\mu(x)=\exp\left(\int P(x)dx\right)$

**通解公式：** $y=\frac{1}{\mu(x)}\left[\int\mu(x)Q(x)dx + C\right]$

**标准解题步骤：**
1. **标准化：** 将方程化为 $\frac{dy}{dx}+P(x)y=Q(x)$ 的形式
2. **求积分因子：** 计算 $\mu(x)=e^{\int P(x)dx}$
3. **乘积分因子：** 两边乘 $\mu(x)$，左边变为 $(\mu(x)y)'$
4. **积分：** 对 $(\mu(x)y)'=\mu(x)Q(x)$ 两边积分
5. **解出y并代入初值**

**常见陷阱：**
- 方程必须先化为标准形式（$y'$ 系数为1）
- 积分时 $\int P(x)dx$ 不需要加常数（取最简单的原函数即可）
- 注意 $P(x)$ 的间断点可能导致解的定义域受限

**做题技巧：**
- 积分因子法是万能方法，但若 $Q(x)=0$（齐次），直接用分离变量更快
- 验证：将解代入原方程检验

**对应习题：** idx 1

### 模块3：一阶齐次方程

**核心概念：** 形如 $\frac{dy}{dx}=F\left(\frac{y}{x}\right)$ 的方程。通过变量代换 $v=\frac{y}{x}$ 化为可分离变量方程。

**标准解题步骤：**
1. **识别：** 验证 $f(x,y)$ 是零次齐次函数，即 $f(tx,ty)=f(x,y)$
2. **代换：** 令 $v=\frac{y}{x}$，则 $y=vx$，$\frac{dy}{dx}=v+x\frac{dv}{dx}$
3. **代入：** 原方程化为 $v+x\frac{dv}{dx}=F(v)$
4. **分离变量：** $\frac{dv}{F(v)-v}=\frac{dx}{x}$
5. **积分并代回：** 积分后用 $v=\frac{y}{x}$ 代回
6. **代入初值确定常数**

**常见陷阱：**
- 别忘了链式法则：$y'=v+xv'$，不要漏掉 $xv'$ 项
- 若 $F(v)=v$，则方程直接变为 $xv'=0$，解为 $v=C$（即 $y=Cx$）

**做题技巧：**
- 判断是否为齐次方程：将 $x,y$ 同时放大 $t$ 倍，看 $dy/dx$ 是否不变
- 如果直接看不出来，试着把右边写成 $y/x$ 的函数

**对应习题：** idx 2

### 模块4：一阶伯努利方程

**核心概念：** 形如 $\frac{dy}{dx}+P(x)y=Q(x)y^n$（$n\neq0,1$）的方程。

**标准解题步骤：**
1. **识别：** 判断方程是否为伯努利形式（右边有 $y^n$ 因子）
2. **变量代换：** 令 $z=y^{1-n}$，则 $\frac{dz}{dx}=(1-n)y^{-n}\frac{dy}{dx}$
3. **化为一阶线性：** 原方程两边乘 $(1-n)y^{-n}$，得 $\frac{dz}{dx}+(1-n)P(x)z=(1-n)Q(x)$
4. **求解线性方程：** 用积分因子法解出 $z(x)$
5. **代回：** $y=z^{1/(1-n)}$
6. **代入初值确定常数**

**常见陷阱：**
- $n=0$ 时退化为线性方程；$n=1$ 时可分离变量，都不需要用此方法
- $y=0$ 可能是奇解（当 $n>0$ 时）
- 代换 $z=y^{1-n}$ 时注意指数运算

**做题技巧：**
- 发现 $y'$ 和 $y$ 的幂次同时出现时，先想到伯努利方程
- 代换后得到的一阶线性方程用积分因子法求解

**对应习题：** idx 3

### 模块5：可降阶的二阶方程（不显含y）

**核心概念：** 形如 $F(x,y',y'')=0$（不显含 $y$）的方程。

**标准解题步骤：**
1. **降阶代换：** 令 $p=y'$，则 $y''=p'=\frac{dp}{dx}$
2. **化为一阶方程：** 原方程变为关于 $p$ 和 $x$ 的一阶方程 $F(x,p,p')=0$
3. **解一阶方程：** 用适当方法解出 $p=p(x,C_1)$
4. **再积分：** $y=\int p(x,C_1)dx + C_2$
5. **代入初值：** 利用 $y(x_0)=y_0$ 和 $y'(x_0)=y_0'$ 确定两个常数

**常见陷阱：**
- 第一步的初值转换：$p(x_0)=y'(x_0)$
- 解 $p$ 时可能分离变量或线性方程，要灵活选择方法
- 第二次积分时不要忘记加 $C_2$

**做题技巧：**
- 看到方程中没有 $y$ 只出现 $y'$ 和 $y''$ 时，立即想到此方法
- 先观察 $p$ 满足的方程类型，再选择解法

**对应习题：** idx 4

### 模块6：可降阶的二阶方程（不显含x）

**核心概念：** 形如 $F(y,y',y'')=0$（不显含 $x$）的方程。

**标准解题步骤：**
1. **降阶代换：** 令 $p=y'$，则 $y''=\frac{dp}{dx}=\frac{dp}{dy}\cdot\frac{dy}{dx}=p\frac{dp}{dy}$
2. **化为一阶方程（以y为自变量）：** $F(y,p,p\frac{dp}{dy})=0$
3. **解一阶方程：** 解出 $p=p(y,C_1)$
4. **再分离变量：** $\frac{dy}{dx}=p(y,C_1)$ → $\frac{dy}{p(y,C_1)}=dx$
5. **积分并代入初值**

**常见陷阱：**
- $y''\neq p'$！这里 $p$ 是 $y$ 的函数，$y''=p\frac{dp}{dy}$
- 若 $p=0$ 是解，需要单独讨论（可能对应常数解）
- 两次积分时注意自变量的转换

**做题技巧：**
- 看到方程中没有 $x$ 只出现 $y,y',y''$ 时，立即想到此方法
- 大多数物理问题中的二阶方程不显含 $x$（如牛顿第二定律不含时间）

**对应习题：** idx 5

### 模块7：二阶常系数齐次线性方程

**核心概念：** 形如 $ay''+by'+cy=0$（$a,b,c$ 为常数，$a\neq0$）的方程。

**核心公式：** 特征方程 $ar^2+br+c=0$

**标准解题步骤：**
1. **写出特征方程：** $ar^2+br+c=0$
2. **求特征根：** $r=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$
3. **根据判别式写通解：**
   - $\Delta>0$（两不等实根 $r_1,r_2$）：$y=C_1e^{r_1x}+C_2e^{r_2x}$
   - $\Delta=0$（重根 $r$）：$y=(C_1+C_2x)e^{rx}$
   - $\Delta<0$（共轭复根 $\alpha\pm i\beta$）：$y=e^{\alpha x}(C_1\cos\beta x+C_2\sin\beta x)$
4. **代入初值确定常数**

**常见陷阱：**
- 特征方程系数与微分方程系数一一对应，别搞错符号
- 复根情形下，$\alpha$ 是实部，$\beta$ 是虚部的绝对值
- $\Delta<0$ 时通解别忘了 $e^{\alpha x}$ 因子

**做题技巧：**
- 先用判别式快速判断类型
- 两个初值条件用来解 $C_1,C_2$，往往需要联立方程组

**对应习题：** idx 6

### 模块8：二阶常系数非齐次—多项式型

**核心概念：** 形如 $ay''+by'+cy=P_n(x)$（$P_n$ 为 $n$ 次多项式）的方程。

**标准解题步骤：**
1. **求齐次通解 $y_h$：** 解特征方程得 $y_h$
2. **设特解形式 $y_p$：**
   - 若 $c\neq0$（$y$ 项存在）：设 $y_p=Q_n(x)$（与右端同次多项式）
   - 若 $c=0,b\neq0$（$y$ 缺席）：设 $y_p=x\cdot Q_n(x)$
   - 若 $c=0,b=0$：设 $y_p=x^2\cdot Q_n(x)$
3. **待定系数：** 将 $y_p$ 代入原方程，比较同次幂系数求未知系数
4. **写通解：** $y=y_h+y_p$
5. **若有初值，代入确定 $C_1,C_2$**

**常见陷阱：**
- 待定多项式次数必须和右端相同（设为一般形式，不要漏项）
- 共振时（0是特征根）需要乘 $x$ 或 $x^2$
- 代入时仔细计算各阶导数

**做题技巧：**
- 设 $y_p$ 时写出含所有低次项的一般形式（如二次：$Ax^2+Bx+C$）
- 比较系数法：代入后按 $x$ 的幂次合并，每个幂次系数为零

**对应习题：** idx 7

### 模块9：二阶常系数非齐次—指数型

**核心概念：** 形如 $ay''+by'+cy=Ae^{\lambda x}$ 的方程。

**标准解题步骤：**
1. **求齐次通解 $y_h$**
2. **设特解形式 $y_p$：**
   - 若 $\lambda$ 不是特征根：$y_p=Be^{\lambda x}$
   - 若 $\lambda$ 是单特征根：$y_p=Bxe^{\lambda x}$
   - 若 $\lambda$ 是重特征根：$y_p=Bx^2e^{\lambda x}$
3. **待定系数：** 代入原方程求 $B$
4. **写通解：** $y=y_h+y_p$

**常见陷阱：**
- 必须检查 $\lambda$ 是否是特征根（共振判断）
- 代入时 $y_p'=B(\lambda x+1)e^{\lambda x}$（乘 $x$ 时），仔细求导

**做题技巧：**
- 快速判断法：将 $y_p=Be^{\lambda x}$ 代入，左边得 $B(a\lambda^2+b\lambda+c)e^{\lambda x}=Ae^{\lambda x}$
- 若 $a\lambda^2+b\lambda+c\neq0$：$B=A/(a\lambda^2+b\lambda+c)$（不共振）
- 若 $a\lambda^2+b\lambda+c=0$：需乘 $x$（共振）

**对应习题：** idx 8

### 模块10：二阶常系数非齐次—三角型

**核心概念：** 形如 $ay''+by'+cy=A\cos\omega x+B\sin\omega x$ 的方程。

**标准解题步骤：**
1. **求齐次通解 $y_h$**
2. **设特解形式：**
   - 若 $\pm i\omega$ 不是特征根：$y_p=C\cos\omega x+D\sin\omega x$
   - 若 $\pm i\omega$ 是特征根（共振）：$y_p=x(C\cos\omega x+D\sin\omega x)$
3. **待定系数：** 代入原方程，比较 $\cos\omega x$ 和 $\sin\omega x$ 的系数
4. **写通解**

**常见陷阱：**
- 即使右端只有 $\sin$ 或只有 $\cos$，特解中两者都要设
- 共振条件：$a\omega^2\mp c=0$ 且 $b=0$（即 $\pm i\omega$ 是特征根）
- 求导时注意乘积法则，特别是乘 $x$ 的共振情形

**做题技巧：**
- 用复指数法也可：设右端为 $Ae^{i\omega x}$，取实部或虚部
- 比较系数时形成关于 $C,D$ 的二元一次方程组

**对应习题：** idx 9

### 模块11：欧拉方程

**核心概念：** 形如 $x^2y''+axy'+by=f(x)$ 的方程（二阶欧拉方程）。

**标准解题步骤：**
1. **变量代换：** 令 $x=e^t$（$x>0$）或 $x=-e^t$（$x<0$），设 $D=\frac{d}{dt}$
2. **转换公式：**
   - $xy'=Dy$
   - $x^2y''=D(D-1)y$
3. **化为常系数方程：** 代入得 $[D(D-1)+aD+b]y=f(e^t)$
4. **求解常系数方程：** 求齐次通解和特解
5. **代回原变量：** $t=\ln|x|$，将 $e^{rt}=x^r$，$t=x^r\ln|x|$ 等代回

**常见陷阱：**
- 必须区分 $x>0$ 和 $x<0$（用 $|x|$ 写结果）
- $D(D-1)y$ 不是 $D^2y$，展开为 $D^2y-Dy$
- 特解中的 $t$ 代回后是 $\ln|x|$

**做题技巧：**
- 齐次解形式：设 $y=x^r$ 代入齐次欧拉方程得特征方程
- 常见齐次解：$x^r$、$x^r\ln|x|$、$x^\alpha\cos(\beta\ln|x|)$ 等

**对应习题：** idx 10

### 模块12：拉普拉斯变换

**核心概念：** 用积分变换将微分方程初值问题化为代数方程。

**核心公式：**
- $\mathcal{L}\{y'\}=sY(s)-y(0)$
- $\mathcal{L}\{y''\}=s^2Y(s)-sy(0)-y'(0)$
- $\mathcal{L}\{e^{at}\}=\frac{1}{s-a}$
- $\mathcal{L}\{t^ne^{at}\}=\frac{n!}{(s-a)^{n+1}}$

**标准解题步骤：**
1. **取拉普拉斯变换：** 对方程两边取变换，代入初值
2. **解出Y(s)：** 得到关于 $Y(s)$ 的代数方程，解出 $Y(s)$
3. **部分分式展开：** 将 $Y(s)$ 分解为简单分式之和
4. **取逆变换：** 利用拉普拉斯变换表逐项求逆
5. **得到解y(t)**

**常见陷阱：**
- $y''$ 的变换涉及 $y(0)$ 和 $y'(0)$ 两个初值，不要遗漏
- 部分分式分解要仔细，特别是重根情形
- 逆变换时注意线性性质

**做题技巧：**
- 熟练掌握常见函数的拉普拉斯变换对
- 卷积定理：$\mathcal{L}\{f*g\}=F(s)G(s)$
- 适用于常系数线性方程，特别是右端有间断或脉冲函数时

**对应习题：** idx 11

### 模块13：幂级数法

**核心概念：** 设解为幂级数 $y=\sum_{n=0}^{\infty}a_nx^n$，代入方程求系数递推关系。

**标准解题步骤：**
1. **设幂级数解：** $y=\sum_{n=0}^{\infty}a_nx^n$，求 $y'$ 和 $y''$
2. **代入方程：** 将所有级数写成同一幂次（通过指标平移）
3. **比较系数：** 各次幂系数为零，得递推公式
4. **求系数：** 用递推公式计算前几项
5. **写解：** $y=a_0y_1(x)+a_1y_2(x)$，其中 $a_0,a_1$ 为任意常数

**常见陷阱：**
- 指标平移时注意求和起点变化
- 递推公式中分母不能为零
- 收敛半径需要另外讨论（但对于线性方程通常无穷大）

**做题技巧：**
- $a_0$ 对应 $y(0)$，$a_1$ 对应 $y'(0)$
- 常点附近幂级数解总是存在且收敛
- 奇点附近需要用 Frobenius 方法（广义幂级数）

**对应习题：** idx 12

### 模块14：常系数线性微分方程组

**核心概念：** 形如 $\vec{x}'=A\vec{x}$（$A$ 为常矩阵）的方程组。

**标准解题步骤：**
1. **写矩阵形式：** $\frac{d}{dt}\vec{x}=A\vec{x}$
2. **求特征值与特征向量：** 解 $\det(A-\lambda I)=0$
3. **构造基解矩阵：**
   - 单实特征值 $\lambda$ + 特征向量 $v$：解为 $e^{\lambda t}v$
   - 复特征值 $\alpha\pm i\beta$：取实部和虚部得两个实解
   - 重特征值且几何重数不足：需广义特征向量，解含 $te^{\lambda t}$ 项
4. **写通解：** 基解的线性组合
5. **代入初值确定常数**

**常见陷阱：**
- 重特征值情形：若几何重数（特征向量个数）小于代数重数，需构造广义特征向量链
- 复特征值：取一个复特征向量，其实部和虚部（乘 $e^{\alpha t}$ 和三角函数）给出两个实解
- 通解中 $C_1,C_2$ 的顺序

**做题技巧：**
- 求 $(A-\lambda I)w=v$ 得广义特征向量
- 重根通解形式：$e^{\lambda t}v$ 和 $e^{\lambda t}(tv+w)$

**对应习题：** idx 13

### 模块15：解的存在唯一性分析

**核心概念：** Picard 存在唯一性定理：若 $f(x,y)$ 在区域上连续且关于 $y$ 满足 Lipschitz 条件，则初值问题有唯一解。

**Lipschitz 条件判断法：**
- 充分条件：$\frac{\partial f}{\partial y}$ 在区域上有界（连续可微则局部满足）
- 不满足时可能出现多解

**标准分析步骤：**
1. **写出 $f(x,y)$：** 从方程中识别右端函数
2. **检查连续性：** $f(x,y)$ 在给定点是否连续
3. **检查 Lipschitz 条件：** 计算 $\frac{\partial f}{\partial y}$，判断是否有界
4. **结论：** 满足条件→解存在唯一；不满足→可能不唯一，尝试构造多个解

**常见陷阱：**
- $\sqrt{y}$ 在 $y=0$ 处偏导无界（经典反例）
- 连续性和 Lipschitz 条件是两个独立的条件

**做题技巧：**
- 构造多解：分离变量求解，注意除以零的情况
- 常见不唯一情形：$y'=y^\alpha$（$\alpha\in(0,1)$）在 $y=0$ 处

**对应习题：** idx 14

---

### 模块16：稳定性分析

**核心概念：** 对自治方程 $\frac{dy}{dt}=f(y)$，平衡解 $y^*$（$f(y^*)=0$）的稳定性由 $f'(y^*)$ 的符号决定。

**核心定理：**
- 若 $f'(y^*)<0$：$y^*$ 渐近稳定
- 若 $f'(y^*)>0$：$y^*$ 不稳定
- 若 $f'(y^*)=0$：需要高阶导数判断

**标准分析步骤：**
1. **求平衡点：** 解 $f(y)=0$
2. **线性化：** 计算每个平衡点的 $f'(y^*)$
3. **判断：** 根据 $f'(y^*)$ 的符号判断稳定性

**常见陷阱：**
- $f'(y^*)=0$ 时不能用线性化判断，需用其他方法
- 注意区分"稳定"和"渐近稳定"

**做题技巧：**
- 一维系统：画 $f(y)$ 的符号图，箭头指向平衡点为稳定
- 多维系统：需用特征值实部判断

**对应习题：** idx 15

### 模块17：奇点分类与相图分析

**核心概念：** 对二维线性系统 $\dot{\vec{x}}=A\vec{x}$，根据特征值分类奇点类型。

**分类标准（按特征值 $\lambda_1,\lambda_2$）：**
| 特征值 | 奇点类型 | 稳定性 |
|--------|----------|--------|
| 两负实根 | 稳定结点 | 渐近稳定 |
| 两正实根 | 不稳定结点 | 不稳定 |
| 一正一负实根 | 鞍点 | 不稳定 |
| 实部为负的共轭复根 | 稳定焦点 | 渐近稳定 |
| 实部为正的共轭复根 | 不稳定焦点 | 不稳定 |
| 纯虚根 | 中心 | 稳定（非渐近） |

**标准分析步骤：**
1. **写系数矩阵 $A$**
2. **求特征值：** 解 $\det(A-\lambda I)=0$
3. **计算 $\tau=\operatorname{tr}(A)$ 和 $\Delta=\det(A)$：** 特征值 $=\frac{\tau\pm\sqrt{\tau^2-4\Delta}}{2}$
4. **分类：** 根据 $\tau,\Delta,\tau^2-4\Delta$ 的符号确定奇点类型
5. **画相图：** 画出特征方向（实特征向量）或旋转方向（复特征值）

**常见陷阱：**
- 中心 vs 焦点：纯虚根（$\tau=0,\Delta>0$）是中心，实部非零是焦点
- 旋转方向可通过在一点计算向量场确定

**做题技巧：**
- 用 $\tau$-$\Delta$ 图快速分类
- 判断旋转方向：计算一点处向量场的叉积符号

**对应习题：** idx 16

### 模块18：正交轨线

**核心概念：** 与给定曲线族中每条曲线正交的曲线族。

**核心公式：** 若原曲线族满足 $y'=f(x,y)$，则正交轨线满足 $y'=-\frac{1}{f(x,y)}$

**标准解题步骤：**
1. **求原曲线族的微分方程：** 对曲线族方程 $F(x,y)=C$ 求导（消去 $C$），得 $y'=f(x,y)$
2. **正交条件：** 正交轨线满足 $y'_{\perp}=-\frac{1}{f(x,y)}$
3. **解正交轨线方程：** 解此一阶微分方程
4. **写正交轨线族**

**常见陷阱：**
- 求原曲线族微分方程时，是求导后消去常数 $C$，不是直接对 $x$ 求导
- 正交条件是斜率乘积为 $-1$：$y'_{\perp}\cdot y'_{\text{原}}=-1$

**做题技巧：**
- 几何解释：梯度方向与曲线正交
- 常见例子：同心圆 ↔ 过原点的直线

**对应习题：** idx 17

### 模块19：物理应用题建模

**核心概念：** 将物理问题转化为微分方程，求解后分析物理意义。

**标准解题步骤：**
1. **分析物理量：** 确定变量（如速度 $v(t)$、位置 $x(t)$ 等）
2. **建立方程：** 利用物理定律（牛顿第二定律等）
3. **确定初值条件：** 从问题中提取初始状态
4. **求解方程：** 选用适当方法
5. **分析物理意义：** 讨论渐近行为、参数影响等

**常见陷阱：**
- 力的方向（正负号）容易搞错
- 阻力通常与速度方向相反
- 极限行为分析：$t\to\infty$ 时的稳态

**做题技巧：**
- 先画出受力分析图
- 注意量纲一致性
- 极限速度 = 合外力为零时的速度

**对应习题：** idx 18

### 模块20：参数讨论

**核心概念：** 分析含参数的微分方程解的性质如何随参数变化。

**标准解题步骤：**
1. **求特征方程/通解（含参数）**
2. **确定参数的分界值：** 使解的性质发生质变的参数值
3. **分情况讨论：** 在每个参数区间内分析解的渐近行为
4. **总结：** 给出稳定性或渐近性与参数的关系

**常见陷阱：**
- 分界点处需要单独讨论（往往是重根或纯虚根情形）
- 不仅考虑实部符号，还要考虑振荡与否

**做题技巧：**
- 画出参数轴，标出分界点，逐段分析
- 关键指标：特征根的实部符号决定衰减/增长

**对应习题：** idx 19

## 第二部分：通用解题方法论

### 如何识别题型

| 方程特征 | 题型 | 解法 |
|----------|------|------|
| 一阶，可写为 $f(x)g(y)$ | 可分离变量 | 直接分离积分 |
| 一阶，$y'+P(x)y=Q(x)$ 形式 | 一阶线性 | 积分因子法 |
| 一阶，$y'=F(y/x)$ 形式 | 齐次方程 | 令 $v=y/x$ |
| 一阶，$y'+P(x)y=Q(x)y^n$ | 伯努利方程 | 令 $z=y^{1-n}$ |
| 二阶，不显含 $y$ | 可降阶（不显含y） | 令 $p=y'$ |
| 二阶，不显含 $x$ | 可降阶（不显含x） | 令 $p=y'$, $y''=p\frac{dp}{dy}$ |
| 二阶，常系数，右端=0 | 齐次线性 | 特征方程 |
| 二阶，常系数，右端=多项式 | 非齐次（多项式型） | 待定系数法 |
| 二阶，常系数，右端=$Ae^{\lambda x}$ | 非齐次（指数型） | 待定系数法 |
| 二阶，常系数，右端=三角函数 | 非齐次（三角型） | 待定系数法 |
| 二阶，$x^2y''+axy'+by=f(x)$ | 欧拉方程 | 令 $x=e^t$ |
| 方程组，$\vec{x}'=A\vec{x}$ | 常系数方程组 | 特征值+特征向量 |

### 如何选择方法

1. **先看阶数：** 一阶方程方法有限，二阶方程先看是否可降阶
2. **再看系数：** 常系数用特征方程法，变系数考虑欧拉方程或幂级数法
3. **最后看右端（非齐次）：** 根据右端形式选待定系数法的特解形式
4. **有初值时：** 拉普拉斯变换特别高效
5. **定性问题：** 存在唯一性→Lipschitz条件；稳定性→线性化；奇点→特征值分类
6. **物理问题：** 先建模再求解，$m\frac{dv}{dt}=\sum F$

### 如何验证结果

1. **代入验证：** 将解代回原方程，检查是否满足
2. **初值验证：** 检查解是否满足给定的初值条件
3. **量纲检查：** 物理问题中检查量纲是否一致
4. **渐近行为：** 检查 $t\to\infty$ 时的行为是否合理
5. **特殊情况：** 检查参数取特殊值时的退化情况
