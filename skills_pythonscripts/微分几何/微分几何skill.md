# 微分几何技能手册

## 高频错误与口径约定（评委反馈 2026-07-07）

- **法曲率符号约定（必须遵守）**：法曲率 $\kappa_n=\mathrm{II}(w,w)/\mathrm{I}(w,w)$ 的符号取决于单位法向量的选取。本题库对圆柱、球面等经典曲面采用**外法向**约定。圆柱面 $\mathbf{r}(u,v)=(\cos v,\sin v,u)$ 取外法向 $\mathbf{n}=(\cos v,\sin v,0)$，此时 $L=\mathbf{r}_{uu}\cdot\mathbf{n}=0$、$N=\mathbf{r}_{vv}\cdot\mathbf{n}=-1$：沿 $u$ 方向（直母线）$\kappa_n=0$，**沿 $v$ 方向（圆周）$\kappa_n=-1$**（曲面向外法向的反方向弯曲，取负）。
- 若按 $\mathbf{n}=\mathbf{r}_u\times\mathbf{r}_v/|\mathbf{r}_u\times\mathbf{r}_v|$ 算出的是内法向（指向轴线/球心），必须翻转法向（等价于结果变号）后再报答案。
- 高斯曲率 $K=\kappa_1\kappa_2$ 与法向选取无关；平均曲率 $H$ 和单个方向的 $\kappa_n$ 随法向翻转而变号——答题时先写明所取法向再给数值。

## 1 曲线弧长

### 核心定义定理

**曲线弧长**是曲线最基本的几何量。对于参数曲线 `r(t) = (x(t), y(t), z(t))`, `t ∈ [a, b]`, 弧长为切向量模长在区间上的积分。

弧长参数化（自然参数化）是微分几何的核心工具——以弧长 `s` 为参数时，切向量恒为单位向量，Frenet 公式形式最简洁。

### 核心公式

| 情形 | 公式 |
|------|------|
| 空间曲线 `r(t)` | `L = ∫_a^b |r'(t)| dt` |
| 平面曲线 `r(t) = (x(t), y(t))` | `L = ∫_a^b √(x'(t)² + y'(t)²) dt` |
| 弧长参数 `s(t)` | `s(t) = ∫_{t₀}^t |r'(τ)| dτ` |

### 标准化分步解题流程

1. 计算切向量 `r'(t)`
2. 计算切向量模长 `|r'(t)| = √(x'(t)² + y'(t)² + z'(t)²)`
3. 对模长在区间 `[a, b]` 上积分

### 考试高频陷阱

- 忘记开平方根：`|r'|` 是各分量平方和的算术平方根
- 忽略被积函数的绝对值处理（如 `cos u` 在积分区间变号）

### 应试解题技巧

- 先简化被积函数再积分：常见三角函数平方和形式 `sin²t + cos²t = 1`
- 螺旋线 `(cos t, sin t, t)` 的弧长直接为 `√2 · 区间长度`

## 2 平面曲线曲率

### 核心定义定理

**曲率** `κ` 刻画曲线偏离直线的程度。几何意义：曲率是切向量方向角对弧长的变化率，也是密切圆半径的倒数。

### 核心公式

| 情形 | 公式 |
|------|------|
| 平面曲线 `r(t) = (x(t), y(t))` | `κ = |x'y'' - x''y'| / (x'² + y'²)^(3/2)` |
| 函数图 `y = f(x)` | `κ = |f''(x)| / (1 + (f'(x))²)^(3/2)` |
| 极坐标 `r = r(θ)` | `κ = |r² + 2(r')² - rr''| / (r² + (r')²)^(3/2)` |

### 标准化分步解题流程

1. 计算 `r'(t)` 和 `r''(t)`
2. 计算 `r' × r''`（二维中为标量叉积 `x'y'' - x''y'`）
3. 计算 `|r'|`
4. 代入 `κ = |r' × r''| / |r'|³`

### 考试高频陷阱

- 对弧长参数化曲线，`κ = |r''(s)|`，但一般参数化不能直接用此简化
- 二维叉积是标量 `x₁y₂ - x₂y₁`，注意符号但取绝对值

### 应试解题技巧

- 圆 `(R cos t, R sin t)` 曲率恒为 `1/R`，可作为验算基准
- 记住平面曲线 `κ > 0` 恒成立（取绝对值后）

## 3 Frenet标架与单位切向量

### 核心定义定理

**Frenet 标架** `{T, N, B}` 是空间曲线上的活动正交标架：
- **单位切向量** `T = r'(t) / |r'(t)|`
- **主法向量** `N = T'(t) / |T'(t)|`（指向曲率中心方向）
- **副法向量** `B = T × N`

`T, N, B` 构成右手正交系，是研究空间曲线局部性质的基础。

### 核心公式

- `T(t) = r'(t) / |r'(t)|`
- `B(t) = (r'(t) × r''(t)) / |r'(t) × r''(t)|`
- `N(t) = B(t) × T(t)`

### 标准化分步解题流程

1. 计算切向量 `r'(t)`
2. 计算模长 `|r'(t)|`
3. 单位化得 `T = r'(t) / |r'(t)|`
4. 代入指定参数值

### 考试高频陷阱

- 忘记单位化：直接拿 `r'(t)` 当单位切向量
- `N` 的计算需要求导 `T'` 再单位化，不能直接对 `r''` 单位化

### 应试解题技巧

- `T` 的方向只与 `r'(t)` 的各分量比例有关，可在代入参数值后再单位化
- 向量 `(1,2,3)/√14` 形式中，分母是模长，分子是 `r'` 的分量

## 4 空间曲线曲率

### 核心定义定理

空间曲线在一点处的弯曲程度由**曲率** `κ` 度量。与平面曲线不同，空间曲线的曲率只反映弯曲大小，不包含扭转信息。

### 核心公式

| 情形 | 公式 |
|------|------|
| 一般参数化 `r(t)` | `κ = |r'(t) × r''(t)| / |r'(t)|³` |
| 弧长参数 `r(s)` | `κ = |r''(s)| = |T'(s)|` |
| 二维退化 | `κ = |x'y'' - x''y'| / (x'² + y'²)^(3/2)` |

### 标准化分步解题流程

1. 计算 `r'(t)` 和 `r''(t)`
2. 计算向量叉积 `r' × r''`
3. 计算叉积模 `|r' × r''|`
4. 计算 `|r'(t)|` 并取三次方
5. `κ = |r' × r''| / |r'|³`

### 考试高频陷阱

- 叉积计算后模长取平方根时遗漏系数
- 未注意 `r''` 各分量的符号对叉积结果的影响

### 应试解题技巧

- 叉积用行列式计算：`r' × r'' = det([i, j, k; x', y', z'; x'', y'', z''])`
- 螺旋线 `(a cos t, a sin t, bt)` 的曲率为 `κ = a / (a² + b²)`

## 5 挠率

### 核心定义定理

**挠率** `τ` 度量空间曲线偏离平面的程度——它反映副法向量 `B` 沿曲线的旋转速率。τ ≡ 0 当且仅当曲线为平面曲线。

### 核心公式

| 情形 | 公式 |
|------|------|
| 一般参数化 `r(t)` | `τ = (r' × r'')·r''' / |r' × r''|²` |
| 弧长参数 `r(s)` | `τ = -B'(s)·N(s)` |

### 标准化分步解题流程

1. 依次计算 `r'(t)`、`r''(t)`、`r'''(t)`
2. 计算叉积 `r' × r''`
3. 计算三重积 `(r' × r'')·r'''`
4. 计算 `|r' × r''|²`
5. `τ = (r' × r'')·r''' / |r' × r''|²`

### 考试高频陷阱

- 将三重积与叉积模混淆：分子是标量三重积，分母是叉积模的平方
- 忘记 `r'''` 的符号——对正弦余弦求导三次可能出错

### 应试解题技巧

- 螺旋线 `(a cos t, a sin t, bt)` 的挠率 `τ = b / (a² + b²)`
- 如果 `r''' = 0`（如多项式不超过二次），挠率必为零
- 常数曲率和挠率（螺旋线）是最简单实用的验证模型

## 6 第一基本形式

### 核心定义定理

**第一基本形式**（度量形式）定义曲面上的内积结构，用于计算曲面上曲线的弧长、夹角和曲面面积。

`I = ds² = E du² + 2F du dv + G dv²`

其中 `E = r_u·r_u`，`F = r_u·r_v`，`G = r_v·r_v`。

### 核心公式

- `E = r_u·r_u`
- `F = r_u·r_v`
- `G = r_v·r_v`
- 面积元 `dA = √(EG - F²) du dv`

### 标准化分步解题流程

1. 计算偏导向量 `r_u = ∂r/∂u` 和 `r_v = ∂r/∂v`
2. 计算内积得 `E, F, G`
3. 表达为 `ds² = E du² + 2F du dv + G dv²`

### 考试高频陷阱

- `F` 可能是零（正交参数化），但不要预先假设
- `G` 中 `cos u` 的平方不要漏掉
- 球面参数化中 `E` 和 `G` 量纲一致但形式不同

### 应试解题技巧

- 正交参数化时 `F = 0`，计算量大减
- 球面第一基本形式：`E = R², F = 0, G = R² cos² u`

---

## 7 第二基本形式

### 核心定义定理

**第二基本形式**描述曲面在一点处偏离切平面的程度，即曲面的"弯曲方式"。

`II = L du² + 2M du dv + N dv²`

其中 `L = r_uu·n`，`M = r_uv·n`，`N = r_vv·n`，`n` 为单位法向量。

### 核心公式

- `n = (r_u × r_v) / |r_u × r_v|`
- `L = r_uu · n`
- `M = r_uv · n`
- `N = r_vv · n`

### 标准化分步解题流程

1. 计算 `r_u, r_v`
2. 计算单位法向量 `n = (r_u × r_v) / |r_u × r_v|`
3. 计算二阶偏导 `r_uu, r_uv, r_vv`
4. 分别与 `n` 点积得 `L, M, N`

### 考试高频陷阱

- 忘记对法向量单位化，直接用 `r_u × r_v` 代替 `n`
- `r_uv` 是混合偏导，不等于零不要跳过

### 应试解题技巧

- 若 `r_u` 与 `r_v` 正交，法向量计算简化
- 圆柱面 `(R cos v, R sin v, u)`：`L = 0, M = 0, N = R`

## 8 法曲率

### 核心定义定理

**法曲率** `κ_n` 是曲面在给定切方向上弯曲程度的度量。它是曲面沿该方向的截面曲线的曲率在法向量上的投影。

### 核心公式

- `κ_n = II / I = (L du² + 2M du dv + N dv²) / (E du² + 2F du dv + G dv²)`
- 沿 `u` 方向（`dv = 0`）：`κ_n = L / E`
- 沿 `v` 方向（`du = 0`）：`κ_n = N / G`

### 标准化分步解题流程

1. 计算该方向的第一、第二基本形式系数
2. 确定方向比 `du : dv`
3. 代入 `κ_n = II / I`

### 考试高频陷阱

- 混淆法曲率与主曲率：法曲率依赖于方向，主曲率是极值
- 符号约定：不同教材法曲率符号可能不同

### 应试解题技巧

- 母线的法曲率为零（直纹面特性）
- 截面圆方向法曲率等于主曲率之一

## 9 高斯曲率

### 核心定义定理

**高斯曲率** `K` 是曲面内蕴几何最重要的量。`K = κ₁κ₂`，即两个主曲率的乘积。

高斯绝妙定理：高斯曲率仅由第一基本形式决定（内蕴量）。

### 核心公式

- `K = (LN - M²) / (EG - F²)`（用基本形式系数）
- `K = κ₁ · κ₂`（用主曲率）
- 可展曲面 `⇔ K ≡ 0`

### 标准化分步解题流程

1. 计算第一基本形式系数 `E, F, G`
2. 计算第二基本形式系数 `L, M, N`
3. 代入 `K = (LN - M²) / (EG - F²)`

### 考试高频陷阱

- 忘记分母 `EG - F²`——这是面积元平方，不能遗漏
- `LN - M²` 的符号决定曲面的局部形状

### 应试解题技巧

- `K > 0`：椭圆点（局部凸），`K < 0`：双曲点（马鞍形），`K = 0`：抛物点或平坦点
- 圆柱面 `K = 0`，球面 `K = 1/R²`（常正曲率）

## 10 平均曲率与极小曲面

### 核心定义定理

**平均曲率** `H = (κ₁ + κ₂)/2`，是主曲率的算术平均值。

**极小曲面**是平均曲率恒为零的曲面。悬链面、螺旋面是经典极小曲面。

### 核心公式

- `H = (EN - 2FM + GL) / (2(EG - F²))`
- 极小曲面条件：`H = 0` ⇔ `LG - 2MF + NE = 0`
- `H = (κ₁ + κ₂)/2`

### 标准化分步解题流程

1. 计算第一、第二基本形式系数
2. 代入 `H = (EN - 2FM + GL) / (2(EG - F²))`
3. 化简看是否为零

### 考试高频陷阱

- 误将 `EN - 2FM + GL` 当作分子而遗忘分母
- 忽略 `F` 和 `M` 的非零可能性

### 应试解题技巧

- 验证极小曲面用 `LG - 2MF + NE = 0`，避免分母运算
- 悬链面 `r = (cosh u cos v, cosh u sin v, u)` 是标准的极小曲面例子

## 11 曲面面积

### 核心定义定理

曲面面积通过第一基本形式的面积元积分计算。

### 核心公式

- 面积元 `dA = √(EG - F²) du dv`
- 曲面面积 `A = ∬_D √(EG - F²) du dv`

### 标准化分步解题流程

1. 计算 `E, F, G`
2. 计算 `√(EG - F²)`
3. 确定参数域 `D`
4. 进行二重积分

### 考试高频陷阱

- 积分区域参数范围弄错（球面参数 `u ∈ [-π/2, π/2]`，不是 `[0, π]`）
- 忘记 `cos u` 在 `[-π/2, π/2]` 上非负，不需要绝对值

### 应试解题技巧

- 球面面积 `4πR²` 可作为验算基准
- 先对简单变量积分（先 `v` 后 `u`）

## 12 测地线与测地曲率

### 核心定义定理

**测地线**是曲面上"最直"的曲线——其测地曲率 `κ_g ≡ 0`。球面上的大圆、圆柱面上的螺旋线都是测地线。

**测地曲率** `κ_g` 是曲线曲率向量在曲面切平面上的投影的长度。

### 核心公式

- 测地线方程：`κ_g = 0`
- `κ_g = |(r' × r'') · n| / |r'|³`（曲面上曲线的测地曲率公式）

### 标准化分步解题流程

1. 验证曲线是否在曲面上
2. 计算曲线的 `r'`、`r''` 和曲面的法向量 `n`
3. 计算 `κ_g` 或判断是否为零

### 考试高频陷阱

- 测地曲率为零不意味着曲线曲率为零（大圆在球面上曲率非零但测地曲率为零）
- 测地线≠最短路径（局部最短是测地线，但测地线不一定最短）

### 应试解题技巧

- 球面的大圆（过球心的平面与球面交线）一定是测地线
- 测地线方程验证困难时，利用对称性判断

## 13 主曲率

### 核心定义定理

**主曲率** `κ₁, κ₂` 是法曲率在两个主方向上的极值。主曲率满足 `K = κ₁κ₂`，`H = (κ₁ + κ₂)/2`。

**脐点**：`κ₁ = κ₂` 的点（球面上的所有点均为脐点）。

### 核心公式

- 由 `(EG - F²)κ² - (EN - 2FM + GL)κ + (LN - M²) = 0` 求解
- 或用 Weingarten 变换矩阵的特征值

### 标准化分步解题流程

1. 计算基本形式系数
2. 构建 `κ` 的二次方程
3. 求解得 `κ₁, κ₂`
4. 验证 `K = κ₁κ₂` 和 `H = (κ₁+κ₂)/2`

### 考试高频陷阱

- 忘记二次项系数 `(EG - F²)` 可能不为1
- 特征值方向与坐标线方向不一致时混淆

### 应试解题技巧

- 在脐点处，任意方向都是主方向
- 计算主曲率优先用公式，避免直接解二次方程

---

## 14 渐近线与双曲点

### 核心定义定理

**渐近方向**是法曲率为零的切方向。**渐近曲线**是每点切方向为渐近方向的曲线。

**双曲点** (`K < 0`) 存在两个不同的渐近方向。**椭圆点** (`K > 0`) 无实渐近方向。**抛物点** (`K = 0`) 有一个渐近方向。

### 核心公式

- 渐近方向条件：`L du² + 2M du dv + N dv² = 0`
- 双曲点判定：`LN - M² < 0` ⇔ `K < 0`

### 标准化分步解题流程

1. 计算 `L, M, N`
2. 写出 `L du² + 2M du dv + N dv² = 0`
3. 求解 `du:dv` 的比例关系

### 考试高频陷阱

- 将第一基本形式与第二基本形式混淆
- 渐近方向是切方向，不是曲面上的曲线

### 应试解题技巧

- 双曲抛物面 `z = x² - y²` 在原点处 `K = -4`，渐近方向为 `du = ±dv`
- 使用对称性快速求出渐近方向

## 15 挠率与平面曲线判定

### 核心定义定理

曲线为平面曲线 ⇔ 挠率恒为零（τ ≡ 0）。几何意义：副法向量 `B` 为常向量，曲线全部位于其密切平面内。

### 核心公式

- τ ≡ 0 ⇒ 曲线是平面曲线
- 平面方程由曲线上三点或由 `r(t)` 各分量关系确定

### 标准化分步解题流程

1. 计算 `r'(t)`, `r''(t)`, `r'''(t)`
2. 计算 `τ = (r' × r'')·r''' / |r' × r''|²`
3. 若 τ ≡ 0，观察或求解满足所有点的线性关系 `ax + by + cz = d`

### 考试高频陷阱

- 只验证了一个点的挠率为零，而不是所有点
- 多项式曲线 `r''' = 0` 但 `r' × r''` 也可能为零（退化情况）

### 应试解题技巧

- `r''' ≡ 0` 且 `r' × r''` 不恒为零时，τ ≡ 0 得证
- 找出曲线上三个不共线的点确定平面

---

## 16 Frenet-Serret公式

### 核心定义定理

Frenet-Serret 公式是空间曲线理论的核心微分方程：

```
T' = κN
N' = -κT + τB
B' = -τN
```

### 核心公式

对弧长参数 `s`：
- `T'(s) = κ(s) N(s)`
- `N'(s) = -κ(s) T(s) + τ(s) B(s)`
- `B'(s) = -τ(s) N(s)`

### 标准化分步解题流程

1. 验证 `|r'(s)| = 1`（确认弧长参数化）
2. `κ = |r''(s)|`
3. `T = r'`, `N = r''/κ`, `B = T × N`
4. `τ = (r' × r'')·r''' / κ²`

### 考试高频陷阱

- 公式只对弧长参数有效，一般参数需要变换
- 符号记忆：`N'` 中有 `-κT` 和 `+τB`

### 应试解题技巧

- 螺旋线 `(a cos s, a sin s, bs)` 对弧长参数，`κ = a`，`τ = b`（当 `a² + b² = 1`）
- 记忆口诀：T 转 N，N 转回 T 并朝 B，B 转回 N

## 17 可展曲面

### 核心定义定理

**可展曲面**是高斯曲率恒为零的直纹面。可展曲面可以展开为平面而不改变度量。

可展曲面包含三类：柱面、锥面、切线曲面（空间曲线的切线曲面）。

### 核心公式

- 可展曲面 ⇔ `K ≡ 0`
- 可展曲面 ⇔ 与平面局部等距

### 标准化分步解题流程

1. 计算 `E, F, G` 和 `L, M, N`
2. 计算 `K = (LN - M²)/(EG - F²)`
3. 验证 `K = 0`

### 考试高频陷阱

- 直纹面不一定可展（单叶双曲面是直纹面但不可展）
- `K = 0` 还需要曲面是直纹面才称为可展曲面

### 应试解题技巧

- 圆柱面、锥面是典型的可展曲面
- `LN - M² = 0` 说明高斯曲率为零

## 18 等距对应

### 核心定义定理

两个曲面间的**等距对应**（等距变换）保持曲面上曲线的长度。**局部等距对应** ⇔ 存在参数化使第一基本形式相同。

等距对应保持高斯曲率不变（高斯绝妙定理）。

### 核心公式

- 等距 ⇔ 在适当的参数化下 `E₁ = E₂, F₁ = F₂, G₁ = G₂`
- 等距 ⇒ `K₁ = K₂`（但反之不成立）

### 标准化分步解题流程

1. 计算两个曲面的第一基本形式系数
2. 比较是否能在某种参数化下得到相同的 `E, F, G`
3. 得出结论

### 考试高频陷阱

- 等距对应不保持第二基本形式（弯曲改变但度量不变）
- 全局等距比局部等距要求更严格

### 应试解题技巧

- 平面与圆柱面局部等距：两者均可用 `du² + dv²` 表示
- 实际例子：将纸张卷成圆柱是等距变换

## 19 Gauss-Bonnet定理

### 核心定义定理

Gauss-Bonnet 定理是微分几何最深刻的定理之一，它将曲面的曲率与拓扑联系起来：

```
∬_D K dA + ∫_{∂D} κ_g ds + Σ (π - 内角) = 2π χ(D)
```

其中 `χ(D)` 是区域 `D` 的 Euler 示性数。

### 核心公式

- 球面三角形（三条大圆弧围成）：`S = R²(α + β + γ - π)`
- 球面 Gauss 曲率：`K = 1/R²`（常值）

### 标准化分步解题流程

1. 确定区域 `D` 和边界
2. 计算各分量：`∬K dA`、`∫κ_g ds`、转角项
3. 代入 Gauss-Bonnet 公式求解未知量

### 考试高频陷阱

- 外角 = `π - 内角`，不要混淆内外角
- 球面三角形边是大圆弧（测地线），测地曲率为零

### 应试解题技巧

- 球面三角形面积公式 `S = R²(α+β+γ-π)` 是高频考点
- 球面正三角形 `α=β=γ=π/2` 占球面 `1/8` 面积

## 20 极小曲面判定

### 核心定义定理

**极小曲面**是平均曲率恒为零 `H ≡ 0` 的曲面。物理上对应"肥皂膜"的形状。

### 核心公式

| 条件 | 表达 |
|------|------|
| `H = 0` | `(EN - 2FM + GL) / [2(EG - F²)] = 0` |
| 等价条件 | `LG - 2MF + NE = 0` |

### 标准化分步解题流程

1. 计算 `E, F, G, L, M, N`
2. 代入 `LG - 2MF + NE` 化简
3. 验证结果为零

### 考试高频陷阱

- 仅用一个点验证不够，需要在整个曲面上 `H ≡ 0`
- `LG - 2MF + NE = 0` 中注意 `M` 和 `F` 的系数是 **-2**

### 应试解题技巧

- 优先验证 `LG - 2MF + NE = 0`，避免复杂的除法运算
- 悬链面和螺旋面是经典的极小曲面

## 21 通用解题方法论

### 题型识别方法

| 题目特征 | 对应知识点 | 优先工具 |
|---------|-----------|---------|
| 求弧长、长度 | 弧长计算 | `|r'|` 积分 |
| 求曲率 κ | 曲率公式 | `|r'×r''|/|r'|³` |
| 求挠率 τ | 挠率公式 | `(r'×r'')·r'''/|r'×r''|²` |
| 求切/法/副法向量 | Frenet 标架 | 单位化 `r'`, 叉积 |
| 求 E,F,G | 第一基本形式 | `r_u·r_u` 等 |
| 求 L,M,N | 第二基本形式 | `r_uu·n` 等 |
| 求法曲率 | 法曲率公式 | `II/I` |
| 求高斯曲率 K | 高斯曲率 | `(LN-M²)/(EG-F²)` |
| 求平均曲率 H | 平均曲率 | `(EN-2FM+GL)/(2(EG-F²))` |
| 求面积 | 曲面面积 | `∬√(EG-F²)` |
| 判断极小曲面 | `H=0` | `LG-2MF+NE=0` |
| 测地线问题 | 测地曲率 | `κ_g=0` |
| 主曲率问题 | 主曲率方程 | 二次方程法 |
| 渐近方向 | 渐近曲线 | `Ldu²+2Mdudv+Ndv²=0` |
| 可展曲面 | `K=0` | `LN-M²=0` |
| 等距对应 | 第一基本形式 | 比较 `E,F,G` |
| 球面三角形 | Gauss-Bonnet | `S=R²(α+β+γ-π)` |

### 解题工具选择

1. **向量法**：曲线问题优先使用向量叉积和点积
2. **参数化**：曲面问题先完成参数化，再计算偏导
3. **正交化**：设法选取正交参数化使 `F = 0` 简化计算
4. **对称性**：利用曲面对称性减少计算量

### 结果核验思路

1. **量纲检查**：曲率量纲为 `1/长度`，弧长为 `长度`，面积为 `长度²`
2. **特殊值验证**：取退化情形的已知结果对比
3. **符号一致性**：`K` 的符号应与曲面形状定性一致
4. **经典公式验证**：螺旋线 `κ = a/(a²+b²)`，`τ = b/(a²+b²)`
