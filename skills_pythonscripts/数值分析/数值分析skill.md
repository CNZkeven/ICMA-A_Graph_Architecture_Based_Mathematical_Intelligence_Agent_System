## 最终答案字段契约

- 不得编造题目未给出的节点、步长、初值或矩阵；所有中间节点和权重必须来自题面或本手册/验证脚本。
- 数值积分题必须写出步长、节点、公式名和求和表达式；题目要求保留精确形式时，优先输出标准加权括号形式。
- Newton/迭代题必须同时给出迭代公式、指定迭代值和精确分数/小数，不得只给近似根。
- 数值微分题必须列出近似值、精确值和绝对误差表达式。
- 条件数/稳定性/ODE 数值解题必须列出范数或放大因子、稳定区间、临界步长和结论。

## 高频错误与口径约定（评委反馈 2026-07-07）

- **绝对稳定区间端点口径**：显式 Euler 对 $y'=\lambda y$（$\lambda<0$）的绝对稳定条件为 $|1+h\lambda|<1$，即 $0<h(-\lambda)<2$——按"渐近稳定"口径**端点取严格不等号**（$h(-\lambda)=2$ 时解振荡不衰减，仅"有界"）。答题默认写严格不等式 $0<h\lambda<2$（记 $\lambda$ 为衰减率的正数记法时），并注明所用口径。
- **最小二乘拟合必须用正规方程独立复核**：直线拟合 $y=a+bx$ 时逐步写出 $n,\sum x_i,\sum y_i,\sum x_i^2,\sum x_iy_i$，解 $\begin{cases}na+b\sum x_i=\sum y_i\\ a\sum x_i+b\sum x_i^2=\sum x_iy_i\end{cases}$，代回两条方程验证；不要跳步直接给系数。

## 1. 误差分析

### 核心定义

| 概念 | 定义 |
|------|------|
| **绝对误差** | 近似值 $x^*$ 与真值 $x$ 之差：$e = x^* - x$ |
| **相对误差** | 绝对误差与真值之比：$\delta = \dfrac{x^* - x}{x}$ 或 $\delta = \dfrac{|x^* - x|}{|x|}$ |
| **有效数字** | 从第一个非零数字起数到误差发生位的位数 |

### 核心公式

$$e = x^* - x \quad \text{(绝对误差)}$$
$$\delta = \frac{e}{x} \quad \text{(相对误差)}$$

### 标准化解题流程

1. 明确近似值和真值
2. 计算绝对误差 $e = x^* - x$
3. 计算相对误差 $\delta = e / x$
4. 根据需要转换为百分比或有效数字表示

### 考试高频陷阱

- 相对误差的分母是真值而非近似值
- 真值未知时用近似值代替需注明
- 绝对误差和相对误差均有正负号

### 应试技巧

- 有理数近似无理数时建议保留分数形式
- 评估误差量级时用科学计数法比较

## 2. 插值法

### 2.1 Lagrange插值

#### 核心定义定理

- 对 $n+1$ 个互异节点 $(x_i, f(x_i))$，存在唯一 $n$ 次多项式 $P_n(x)$ 满足 $P_n(x_i) = f(x_i)$
- **插值基函数**：$L_k(x) = \displaystyle\prod_{j \neq k} \frac{x - x_j}{x_k - x_j}$

#### 核心公式

$$P_n(x) = \sum_{k=0}^{n} f(x_k) L_k(x)$$

**两点线性插值** ($n=1$)：
$$P_1(x) = f(x_0)\frac{x-x_1}{x_0-x_1} + f(x_1)\frac{x-x_0}{x_1-x_0}$$

**三点二次插值** ($n=2$)：
$$P_2(x) = f(x_0)\frac{(x-x_1)(x-x_2)}{(x_0-x_1)(x_0-x_2)} + f(x_1)\frac{(x-x_0)(x-x_2)}{(x_1-x_0)(x_1-x_2)} + f(x_2)\frac{(x-x_0)(x-x_1)}{(x_2-x_0)(x_2-x_1)}$$

#### 标准化解题流程

1. 确定节点数量和节点值
2. 逐个构造基函数 $L_k(x)$
3. 加权求和得到 $P_n(x)$
4. 代入指定 $x$ 求近似值

#### 高频陷阱

- 分母注意符号（$x_k - x_j$ 与 $x_j - x_k$ 方向不可混淆）
- 节点需互异
- 高次插值可能产生Runge现象

#### 应试技巧

- 对对称节点通常可先观察函数值规律简化计算
- 检查插值条件：$P_n(x_i) = f(x_i)$ 用于验证结果

---

### 2.2 插值余项与误差分析

#### 核心公式

**Lagrange插值余项**（若 $f \in C^{n+1}[a,b]$）：
$$R_n(x) = f(x) - P_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} \prod_{i=0}^{n} (x - x_i)$$
其中 $\xi$ 在 $x$ 和节点之间的区间内。

**线性插值余项**（$n=1$）：
$$R_1(x) = \frac{f''(\xi)}{2} (x - x_0)(x - x_1)$$

#### 标准化解题流程

1. 确定插值节点 $x_0, x_1, \dots, x_n$
2. 计算 $(n+1)$ 阶导数 $f^{(n+1)}(\xi)$
3. 乘积 $\prod (x - x_i)$ 在指定点求值
4. 结合导数界得到误差范围

#### 高频陷阱

- $\xi$ 在区间内不可确定具体位置，只能用最大模估计
- 余项公式要求函数有足够光滑性

## 3. 数值积分

### 3.1 中点公式

#### 核心公式

$$\int_a^b f(x)\,dx \approx (b-a) f\!\left(\frac{a+b}{2}\right)$$

**误差**（对二次函数精确）：
$$E_M = \frac{(b-a)^3}{24} f''(\xi), \quad \xi \in (a,b)$$

#### 标准化解题流程

1. 取区间中点 $c = (a+b)/2$
2. 计算 $f(c)$
3. 积分近似 = $(b-a) \times f(c)$
4. 比较精确值求误差

---

### 3.2 梯形公式

#### 核心公式

$$\int_a^b f(x)\,dx \approx \frac{b-a}{2}[f(a) + f(b)]$$

**误差**：
$$E_T = -\frac{(b-a)^3}{12} f''(\xi), \quad \xi \in (a,b)$$

#### 标准化解题流程

1. 求端点函数值 $f(a), f(b)$
2. 代入梯形公式
3. 如需误差分析，求 $f''(\xi)$ 的最大模

#### 应试技巧

- 代数精度至少为 1（对一次多项式精确）
- 凹函数 $f''(x) > 0$ 时梯形公式高估积分，凸函数低估

---

### 3.3 Simpson公式

#### 核心公式

$$\int_a^b f(x)\,dx \approx \frac{b-a}{6}\left[f(a) + 4f\!\left(\frac{a+b}{2}\right) + f(b)\right]$$

**误差**：
$$E_S = -\frac{(b-a)^5}{2880} f^{(4)}(\xi), \quad \xi \in (a,b)$$

#### 核心性质

- **代数精度为 3**（对三次及以下多项式精确成立）
- 三点二阶Newton-Cotes公式

#### 标准化解题流程

1. 求端点及中点函数值 $f(a), f(\frac{a+b}{2}), f(b)$
2. 按 $S = \frac{h}{3}[f(a) + 4f(mid) + f(b)]$ 计算（$h = \frac{b-a}{2}$）
3. 与精确值比较验证

---

### 3.4 复化求积公式

#### 核心公式

**复化梯形**（$n$ 等分，$h = \frac{b-a}{n}$）：
$$T_n = \frac{h}{2}\left[f(x_0) + 2\sum_{i=1}^{n-1} f(x_i) + f(x_n)\right]$$

**复化Simpson**（$2n$ 等分，$h = \frac{b-a}{2n}$）：
$$S_n = \frac{h}{3}\left[f(x_0) + 4\sum_{i=1}^{n} f(x_{2i-1}) + 2\sum_{i=1}^{n-1} f(x_{2i}) + f(x_{2n})\right]$$

#### 标准化解题流程

1. 确定 $n$ 和 $h$
2. 枚举所有节点 $x_i$
3. 计算各节点函数值
4. 按公式加权求和

#### 考试高频陷阱

- 注意 $n$ 是子区间数，节点数为 $n+1$
- 复化梯形内部系数均为 2，两端为 1
- 保留中间表达式（如 $e$ 的精确形式）不需求小数

---

### 3.5 Romberg外推加速

#### 核心公式

Romberg序列递推（Richardson外推）：
$$T_{m}^{(k)} = \frac{4^k T_{m+1}^{(k-1)} - T_{m}^{(k-1)}}{4^k - 1}$$

常用形式（梯形外推得Simpson）：
$$S = \frac{4T(h/2) - T(h)}{3}$$

#### 标准化解题流程

1. 计算 $T(h)$（步长 $h$ 的复化梯形值）
2. 计算 $T(h/2)$（步长减半）
3. 外推得更高精度近似：$S = (4T(h/2) - T(h))/3$
4. 可反复外推直至满意精度

#### 应试技巧

- $T(h)$ 与 $T(h/2)$ 均用精确分数表示避免舍入
- 一次外推得到与Simpson公式同阶精度

## 4. 方程求根

### 4.1 Newton迭代法

#### 核心公式

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

**迭代函数**：$\varphi(x) = x - \dfrac{f(x)}{f'(x)}$

**收敛阶**：若 $f(x^*) = 0$ 且 $f'(x^*) \neq 0$，则Newton法**二阶收敛**（平方收敛）：
$$\lim_{n\to\infty} \frac{|e_{n+1}|}{|e_n|^2} = \frac{|f''(x^*)|}{2|f'(x^*)|}$$

#### 标准化解题流程

1. 写出 $f(x)$ 和 $f'(x)$
2. 选择初值 $x_0$（尽量靠近根）
3. 迭代计算 $x_{n+1} = x_n - f(x_n)/f'(x_n)$
4. 直到 $|x_{n+1} - x_n|$ 小于容忍度

#### 高效应试技巧

- 求 $\sqrt{a}$ 可构造 $f(x) = x^2 - a$，迭代格式简化为 $x_{n+1} = \dfrac{x_n + a/x_n}{2}$
- 每次迭代有效数字约翻倍（二阶收敛的显著特征）
- 可用分数精确计算以避免舍入误差传播

#### 高频陷阱

- $f'(x_n) \approx 0$ 时迭代发散
- 初值选取不当可能收敛到非预期根
- 重根时降为一阶收敛

---

### 4.2 不动点迭代

#### 核心定义

将方程 $f(x)=0$ 改写为等价形式 $x = \varphi(x)$，构造迭代：
$$x_{n+1} = \varphi(x_n)$$

#### 收敛性定理（压缩映射原理）

若 $\varphi(x)$ 在 $[a,b]$ 上可微，且：
1. $\varphi([a,b]) \subseteq [a,b]$
2. $|\varphi'(x)| \leq L < 1$ 对 $\forall x \in [a,b]$

则迭代收敛到唯一不动点 $x^*$，且：
$$|x_n - x^*| \leq \frac{L^n}{1-L} |x_1 - x_0|$$

**渐近收敛因子**：$\rho = |\varphi'(x^*)|$

#### 标准化解题流程

1. 将方程改写为 $x = \varphi(x)$
2. 验证 $\varphi$ 在区间上满足压缩映射条件
3. 选择初值 $x_0$ 开始迭代
4. 计算 $x_1, x_2, x_3, \dots$
5. 用 $|x_{n+1} - x_n|/|x_n - x_{n-1}|$ 估计收敛因子

#### 高频陷阱

- 不同 $\varphi$ 选法收敛性不同
- 局部收敛条件 $|\varphi'(x^*)| < 1$ 只在 $x^*$ 附近成立
- 线性收敛（一阶）比Newton法慢

## 5. 线性方程组直接法

### 5.1 Gauss消去法与列主元法

#### 核心定义

**顺序Gauss消去**：通过初等行变换将增广矩阵化为上三角阵，再回代求解。

**列主元Gauss消去**：每步消元前选取当前列绝对值最大的元素作为主元（行交换），提高数值稳定性。

#### 核心步骤

1. **选主元**：在当前列 $k$ 下方选择 $|A_{ik}|$ 最大的行 $i$，交换第 $k$ 行与第 $i$ 行
2. **消元**：对 $i > k$，$m_{ik} = A_{ik} / A_{kk}$，$A_{i,j} \gets A_{i,j} - m_{ik} A_{k,j}$，$b_i \gets b_i - m_{ik} b_k$
3. **回代**：$x_n = b_n/A_{nn}$，$x_i = (b_i - \sum_{j=i+1}^n A_{ij}x_j)/A_{ii}$

#### 应试技巧

- 列主元可避免小主元导致的数值不稳定
- 解唯一时行列式非零
- 分数计算保持精确性

---

### 5.2 LU分解

#### 核心定义

将 $A$ 分解为 $A = LU$，其中 $L$ 为单位下三角阵，$U$ 为上三角阵（Doolittle形式）。

#### 核心公式

**Doolittle分解算法**（$k=1,\dots,n$）：
$$u_{kj} = a_{kj} - \sum_{r=1}^{k-1} l_{kr} u_{rj}, \quad j = k,\dots,n$$
$$l_{ik} = \frac{1}{u_{kk}}\left(a_{ik} - \sum_{r=1}^{k-1} l_{ir} u_{rk}\right), \quad i = k+1,\dots,n$$

#### 解方程组流程

1. 对 $A$ 做LU分解得 $L$ 和 $U$
2. **前代**解 $Ly = b$：$y_i = b_i - \sum_{j=1}^{i-1} l_{ij} y_j$
3. **回代**解 $Ux = y$：$x_i = (y_i - \sum_{j=i+1}^{n} u_{ij} x_j)/u_{ii}$

#### 高频陷阱

- Doolittle分解 $L$ 对角元为 1，Crout分解 $U$ 对角元为 1
- 若某步主元 $u_{kk} = 0$ 则分解失败（需行交换）

#### 应试技巧

- 消元因子直接填入 $L$ 对应位置
- 验证 $A = LU$ 是检查分解正确性的最快方法

## 6. 线性方程组迭代法

### Jacobi迭代法

#### 核心公式

将 $Ax = b$ 分裂为 $A = D - L - U$（$D$ 对角阵，$-L$ 严格下三角，$-U$ 严格上三角）：
$$x^{(k+1)} = D^{-1}(L+U)x^{(k)} + D^{-1}b$$

**迭代矩阵**：
$$T_J = D^{-1}(L+U)$$

#### 收敛性判定

**充要条件**：迭代矩阵的谱半径 $\rho(T_J) < 1$

**充分条件**（对角占优）：若 $A$ 严格对角占优，则Jacobi迭代收敛

#### 谱半径计算

$$\rho(T) = \max\{|\lambda| : \det(T - \lambda I) = 0\}$$

- 若 $\rho < 1$，迭代收敛（线性收敛速度）
- 收敛速度由 $\rho$ 决定，$\rho$ 越小收敛越快

## 7. 最小二乘拟合

### 核心定义

给定数据点 $(x_i, y_i), i=1,\dots,m$，求参数 $a,b$ 使残差平方和最小：
$$\min_{a,b} \sum_{i=1}^m (y_i - (a + bx_i))^2$$

### 核心公式

**正规方程组**（直线拟合 $y = a + bx$）：
$$\begin{pmatrix} m & \sum x_i \\ \sum x_i & \sum x_i^2 \end{pmatrix}
\begin{pmatrix} a \\ b \end{pmatrix} =
\begin{pmatrix} \sum y_i \\ \sum x_i y_i \end{pmatrix}$$

**解**：
$$b = \frac{m\sum x_i y_i - \sum x_i\sum y_i}{m\sum x_i^2 - (\sum x_i)^2}$$
$$a = \frac{\sum y_i - b\sum x_i}{m}$$

**残差平方和**：
$$RSS = \sum_{i=1}^m (y_i - (a + bx_i))^2$$

### 标准化解题流程

1. 计算 $\sum x_i$, $\sum y_i$, $\sum x_i^2$, $\sum x_i y_i$
2. 代入正规方程组
3. 解线性方程组得 $a, b$
4. 计算残差平方和验证拟合质量

### 应试技巧

- 系数用分数或小数均可，考试推荐分数保持精确
- 可用矩阵形式 $\Phi^T\Phi \beta = \Phi^T y$ 统一表述

## 8. 数值微分

### 核心公式

**三点中心差分公式**：
$$f'(x) \approx \frac{f(x+h) - f(x-h)}{2h}$$

**截断误差**：$O(h^2)$

**前向差分**：
$$f'(x) \approx \frac{f(x+h) - f(x)}{h}, \quad O(h)$$

**后向差分**：
$$f'(x) \approx \frac{f(x) - f(x-h)}{h}, \quad O(h)$$

### 标准化解题流程

1. 确定 $x$、$h$ 和 $f(x)$ 表达式
2. 计算 $f(x+h)$ 和 $f(x-h)$
3. 代入中心差分公式
4. 比较精确导数求误差

### 高频陷阱

- $h$ 过小时因相近数相减产生严重舍入误差
- $h$ 过大时截断误差增大
- 最优步长 $h \approx \sqrt[3]{\epsilon/M}$（$\epsilon$ 为机器精度，$M$ 为三阶导数界）

## 9. 矩阵范数与条件数

### 核心定义

**Frobenius范数**（矩阵元素的平方和开方）：
$$\|A\|_F = \sqrt{\sum_{i=1}^m\sum_{j=1}^n |a_{ij}|^2}$$

**1-范数（列范数）**：
$$\|A\|_1 = \max_{1\leq j\leq n} \sum_{i=1}^m |a_{ij}|$$

**条件数**：
$$\kappa(A) = \|A\| \cdot \|A^{-1}\|$$

### 核心公式

$$\|A\|_F = \sqrt{\sum_i\sum_j a_{ij}^2}$$
$$\kappa_1(A) = \|A\|_1 \|A^{-1}\|_1$$

### 条件数含义

- $\kappa(A)$ 大 $\rightarrow$ **病态矩阵**（微扰导致解剧烈变化）
- $\kappa(A) \approx 1$ $\rightarrow$ **良态矩阵**
- Hilbert矩阵是经典病态矩阵

### 标准化解题流程

1. 计算 Frobenius 范数：各元素平方和开方
2. 计算 1-范数：最大列绝对值之和
3. 求逆矩阵
4. 计算条件数 = $\|A\| \cdot \|A^{-1}\|$

### 高频陷阱

- 不同范数定义的条件数值不同
- 计算逆矩阵注意行列式为零的情况
- $\kappa(A) \geq 1$ 恒成立

## 10. 数值稳定性

### 核心概念

**数值稳定性**：算法对舍入误差的敏感程度。

**灾难性抵消**：两个相近的大数相减导致有效数字大量损失。

### 典型分析

**计算 $e^{-10}$ 的两种方法对比**：

| 方法 | 操作 | 稳定性 |
|------|------|--------|
| A: Taylor级数展开 $\sum (-10)^k/k!$ | 交替大项相消 | **不稳定** |
| B: 计算 $e^{10}$ 再取倒数 $1/e^{10}$ | 无抵消 | **稳定** |

**Taylor级数前5项**：
$$S_5 = \sum_{k=0}^4 \frac{(-10)^k}{k!} = 1 - 10 + 50 - \frac{1000}{6} + \frac{10000}{24} = 291$$

与实际值 $e^{-10} \approx 4.54\times 10^{-5}$ 完全偏离，验证了方法A的不可靠性。

### 应试技巧

- 遇到符号交替且项值很大的级数求和，优先考虑数值稳定性问题
- 避免直接计算差值很小的量
- 通过数学变换提高稳定性（如 $e^{-a} = 1/e^a$）

## 11. 常微分方程数值解

### 显式Euler法

#### 核心公式

对初值问题 $y' = f(t,y),\; y(t_0) = y_0$：
$$y_{n+1} = y_n + h f(t_n, y_n)$$

#### 绝对稳定性

**模型方程**：$y' = -\lambda y$（$\lambda > 0$）

**Euler格式**：
$$y_{n+1} = (1 - h\lambda) y_n$$

**绝对稳定条件**：$|1 - h\lambda| \leq 1$

**稳定区间**：
$$0 < h\lambda < 2$$

**临界步长**：
$$h_{\text{crit}} = \frac{2}{\lambda}$$

### 标准化解题流程

1. 确定 $\lambda$ 和步长 $h$
2. 计算放大因子 $G = 1 - h\lambda$
3. 若 $|G| \leq 1$ 则稳定，否则不稳定
4. 临界步长 = $2/\lambda$

### 应试技巧

- 刚性方程要求步长极小，需用隐式方法
- 稳定区间与 $h\lambda$ 有关，是缩放后的量
- $h\lambda > 2$ 时数值解振荡发散

## 12. 通用解题方法论

### 题型识别方法

| 题型特征 | 所属模块 | 推荐方法 |
|----------|----------|----------|
| "近似值"+"误差/误差" | 误差分析 | 绝对/相对误差公式 |
| 给定节点求多项式 | 插值法 | Lagrange/Newton插值 |
| 计算定积分近似值 | 数值积分 | 梯形/Simpson/复化公式 |
| 方程 $f(x)=0$ 求根 | 方程求根 | Newton法/不动点迭代 |
| 解线性方程组 | 直接法/迭代法 | Gauss消去/LU分解/Jacobi |
| 拟合数据点 | 最小二乘 | 正规方程组 |
| 求导数近似值 | 数值微分 | 中心差分公式 |
| 判断矩阵病态 | 条件数 | 计算 $\kappa(A)$ |
| 分析算法稳定性 | 数值稳定性 | 误差传播分析 |
| ODE初值问题 | 数值解ODE | Euler法/RK方法 |

### 解题工具选择

- **分数运算**优先于小数，避免舍入误差
- **对称结构**利用函数值规律简化计算
- **验证检查**：插值验证节点、分解验证 $A=LU$、拟合验证RSS
- **收敛判据**：迭代法优先看谱半径 $\rho$，其次看对角占优性

### 结果核验思路

1. **量纲检查**：积分值应为正（被积函数为正时）
2. **边界条件**：插值多项式应精确通过给定节点
3. **单调性**：迭代序列应单调趋近极限（单调收敛时）
4. **精确验证**：简单情形可手算一至两步确认
5. **替代方法**：用不同方法交叉验证（如梯形与Simpson比较）
